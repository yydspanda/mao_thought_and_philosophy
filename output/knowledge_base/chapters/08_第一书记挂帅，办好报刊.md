# 第8章 第一书记挂帅，办好报刊

> **摘要**：在1958年大跃进启动和反修正主义斗争的背景下，毛泽东将省级报刊的领导权从宣传部门提升至战略核心，要求省委第一书记亲自挂帅，将其作为统一思想、发动群众、指导工作的首要工具，从而实现对全省政治、经济、思想的全面掌控。

1958年在中央先后召开的南宁会议、成都会议、武昌会议上，毛主席多次谈到省委第一书记要亲自抓好党报、党刊问题。在南宁会议上，毛主席提到他1月12日写给广西省委第一书记刘建勋、省长韦国清的一封信，是谈办好省报问题的。信中说：“送上几份地方报纸，各有特点，是比较编得好的,较为引人看，内容也不错，供你们参考。省报问题是一个极其重要的问题，值得认真研究，同广西日报的编辑们一道，包括版面、新闻、社论、理论、文艺等项。钻进去，想了又想，分析又分析，同各省报纸比较又比较，几个月时间就可以找出一条道路来的。精心写作社论是一项极其重要的任务，你们自己、宣传部长、秘书长、报社总编辑，要共同研究。第一书记挂帅，动手修改一些最重要的社论，是必要的。一张省报，对于全省工作，全体人民,有极大的组织、鼓舞、激励、批判、推动的作用。请你们想一想这个问题，以为如何？”后来，广西一位负责同志告诉我说：毛主席送的几份地方报纸中有河北日报、山西日报等，并且毛主席还当面对刘建勋、韦国清同志评论这些报纸说，山西日报的评论文章包括社论写得比较好，有的很有份量。河北日报的版面编排得比较活泼，报道文章也短小生动。毛主席那样忙，还抽出时间阅读地方报纸，而且那样细致认真，又有比较，又有评论，我们这些当省委第一书记的能不心悦诚服么！毛主席在这次会上还讲：他所以写这封信是要提醒全党，不仅中央纸，而且省级报纸也要认真办好，关键是第一书记挂帅，党要抓紧。在同年3月9日的成都会议上，毛主席又说，我们织和指导工作，主要靠报纸，单是开会，效果有限。3月22毛主席在这次会上的讲话中提出“要提高风格，振作精神”时批评了陈伯达，说他过去死也不肯办一个理论刊物，胆子太小这次才振作精神，下决心办。毛主席还要求各省、市、自治党委也要办自己的理论刊物。各省办的刊物，也要富有理论彩，内容不一定限于本省，宇宙间的一切事情都可以谈。<br><br>为了执行毛主席的这一指示，会后山西省委很快就办了己的理论刊物《前进》。这个刊物的刊名是我报请毛主席题写的一直沿用至今。

## 🧠 深度思考与解读

## 1. 历史场域与战略意图 (Context & Intent)

毛泽东在1958年南宁、成都会议上的讲话，其背景远比“办好报纸”复杂，是其宏大战略布局的关键一环。

- **时局：** 1958年是“大跃进”运动的开端。这是一场旨在以超常规速度实现工业化和共产主义的巨大社会实验。同时，国际上，1956年苏共二十大赫鲁晓夫的“秘密报告”引发了社会主义阵营的思想混乱，毛泽东对苏联的“修正主义”路线高度警惕。国内，1957年的反右运动刚刚结束，思想领域实现了高度统一，为接下来的激进运动扫清了障碍。

- **靶子：** 表面上批评的是陈伯达办刊“胆子太小”，实际上针对的是全党中高级干部中普遍存在的“官僚主义”和“事务主义”倾向。这些干部可能将报纸、刊物视为宣传部门的常规工作，满足于开会、发文件，而没有认识到这是发动群众、统一意志的“精神原子弹”，是决定大跃进成败的战略武器。

- **意图：**
    1.  **为“大跃进”进行极限动员：** 大跃进需要打破一切常规，这无法依靠传统的行政命令。毛泽东意图通过党报党刊，将中央的战略意图和革命热情直接“烧”到基层，绕过层层衰减的官僚体系，形成“一天等于二十年”的强大社会动能。
    2.  **构建反“修正主义”的思想防线：** 毛泽东认为苏联的“变修”始于思想领域的松懈和领导权的旁落。他要求各省办理论刊物，是在全国范围内建立无数个捍卫其思想路线的[[意识形态阵地论]]据点，防止中央出现问题时（如赫鲁晓夫式的突变），地方能有独立的思想阵地进行抵抗，这是深刻的战略远见和不安全感的体现。
    3.  **重塑领导方式：** 他试图将省委书记从“行政官”改造为“思想领袖”和“群众鼓动家”。亲自抓报纸，就是强迫他们从日常琐事中抽离，去思考全局性、方向性的战略问题，学会用思想和舆论来领导工作。

## 2. 核心方法论建模 (Methodology Modeling)

毛泽东在文中提出的方法，可以抽象为两个至今仍极具威力的管理模型：

### 模型一：[[第一书记总编辑法则]] (The First Secretary as Chief Editor Principle)

这是一个关于领导权和组织控制的核心法则，主张领导者必须将组织的“麦克风”牢牢掌握在自己手里。其操作步骤是：

1.  **战略定位 (Positioning):** 明确定义沟通平台（报纸/内网/公众号）的战略作用——“组织、鼓舞、激励、批判、推动”，使其服务于组织的核心目标，而不只是信息发布。
2.  **亲自入局 (Deep Dive):** 领导者必须亲自阅读、研究、对比平台内容（“钻进去，想了又想，分析又分析”）。这不仅是审阅，更是通过内容感知组织脉搏和一线炮火。
3.  **关键干预 (Critical Intervention):** 对于最能体现组织意志和战略方向的内容（“最重要的社论”），第一负责人必须亲自修改、审定。这确保了核心信息的准确性和权威性。
4.  **团队共创 (Team Collaboration):** 组建由第一书记、秘书长、宣传部长、总编辑构成的核心内容策动小组，将领导意志与专业执行紧密结合。

### 模型二：[[意识形态阵地前移模型]] (Ideological Frontline Decentralization Model)

这是一个应对思想僵化和组织板结的战略模型，通过“去中心化”的授权，激活整个组织的思想活力。

1.  **强制建阵 (Mandatory Creation):** 最高层下达指令，要求所有核心业务单元（省委/事业部）必须建立自己的“理论刊物”（内部论坛/学习小组/研究期刊）。
2.  **拓宽疆域 (Broaden Scope):** 鼓励这些地方平台“宇宙间的一切事情都可以谈”，打破部门墙和视野局限，从更高维度思考问题，防止“山头主义”和“本位主义”。
3.  **激活神经末梢 (Activate Nerve Endings):** 该模型将思想建设的责任从总部下放到一线，迫使地方领导者思考根本性、全局性问题，从而在全组织范围内培养战略思维，建立人才梯队。

## 3. 哲学底层逻辑 (Philosophical Root)

这些方法论根植于马克思主义哲学的核心原理：

- **认识论与实践论：** 毛泽东强调“钻进去”、“比较又比较”，这正是《实践论》的核心思想——真知来源于实践，特别是深入的、比较性的实践。他反对脱离一线实际的官僚式领导，主张领导者必须投身于“改造世界”的实践中（办报纸就是一种实践），才能获得正确的认识。

- **矛盾论——抓主要矛盾：** 在1958年复杂的国内外形势下，毛泽东敏锐地抓住了“思想统一”和“组织动员”这个主要矛盾。他认为，只要解决了这个矛盾，生产力发展中的其他次要矛盾（如技术、资源不足）就能通过强大的主观能动性来克服。因此，掌控报刊这一思想工具，就成了解决主要矛盾的“牛鼻子”。

- **上层建筑反作用于经济基础：** 这是历史唯物主义的关键原理。毛泽东极度重视报刊、理论等“上层建筑”的作用，坚信先进的思想和强大的舆论能够极大地促进“经济基础”的变革。他要求第一书记抓报纸，本质上就是运用上层建筑的力量，为大跃进这一经济基础的剧变提供最强的反作用力。

## 4. 现实场景迁移 (Pragmatic Application)

历史的剧本在今天的商业世界中不断重演。当年的“官僚主义”和“思想混乱”，在现代企业中表现为“部门墙林立”、“企业文化稀释”、“战略执行不到位”。

- **场景映射：**
    - **“空心化的企业文化”：** 许多公司的价值观挂在墙上，但从未融入业务决策。CEO忙于看财报和开战略会，却将文化建设、内部沟通完全授权给HR或市场部，导致战略意图在传递中层层失真、变形，这就是当代企业中的“第一书记不看报”。
    - **“创新的孤岛”：** 创新往往被限定在总部的“战略部”或“创新中心”，而广大的业务部门只是被动执行者。这就像只有中央办理论刊物，地方思想僵化一样，无法激发整个组织的创新活力。

- **行动建议：**
    1.  **CEO/管理者应成为“首席布道官”：** 领导者必须将内部沟通视为核心领导力，而非辅助职能。每周亲自撰写一篇内部信、录制一段视频，或在内部论坛上发起一个关于公司战略/文化的深度讨论。用自己的语言，反复、真诚地阐述“我们为何而战”，将战略意图转化为员工能理解、能共鸣的故事。
    2.  **推行“业务单元的OKR哲学反思会”：** 模仿“地方办理论刊物”，要求每个核心业务单元（BU）每季度召开一次“哲学反思会”，不仅是复盘业务数据（KPI/OKR），更要回答“我们这个季度的业务实践，是如何体现/挑战/丰富了公司的核心价值观？”并将洞察分享给全公司。这能有效打破思想的“部门墙”，让文化真正落地。
    3.  **将“叙事能力”纳入高管评估体系：** 在评估一个高管时，除了业务指标，增加一个“组织叙事与动员能力”的权重。评估他/她是否能清晰地将公司战略转化为本团队的使命，并持续有效地激励团队。这正是将[[第一书记总编辑法则]]制度化，确保各级“封疆大吏”都具备思想领导力。
