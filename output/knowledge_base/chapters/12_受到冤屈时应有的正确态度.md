# 第12章 受到冤屈时应有的正确态度

> **摘要**：本章通过毛泽东个人经历，强调了在党内遭受冤屈时，应将逆境转化为学习与实践的动力，坚定对真理和党实事求是原则的信念，从而培养政治韧性。

1956年9月毛主席在“八大”预备会议上讲到一个共产党员在党内受到冤屈时，往往有两种不同的态度。一种是，从此消极不满，一蹶不振；另一种是，把它当作有益的教育，发奋读书，继续寻求真理，坚定信心。第一种态度，显然是错误的，不可取的，应该采取第二种态度。毛主席说，我过去受打击、受处分共有20次之多。人家批评我说，“枪杆子出政权是错误的”，“山上没有马克思主义”，我被封为“狭隘经验主义”，就是说，一点马克思主义也没有了。撤了职，没事干，逼得我只好读书。读书当然就要联系中国革命的具体实践，从陈独秀的右倾机会主义到王明的“左”倾机会主义给中国革命造成的血的教训中，寻求中国革命的正确道路，使我得到很大的益处。<br><br>毛主席说，一个人受到冤屈，就是说，你本来是正确的，但被人封为机会主义者，错误不在你，而在封你为机会主义的人。在这种情况下，你就要相信一条真理，叫做，白的说不黑，黑的说不白。或者叫做，坏的说不好，好的说不坏。要相信我们的党是掌握真理的党，是实事求是的党。过去受到过冤屈的好同志，往往十年二十年甚至更长的时间得不到平反，但是终究是会得到平反的，生前得不到平反，死后也会得到平反昭雪的。否则我们的党就不是讲真理、讲实事求是的党了。曹操、秦始皇、纣王我们还不是给他们恢复名誉了么？<br><br>毛主席的这些话给我的印象特别深刻。“文化大革命”中，我受到林彪、江青反革命集团的打击迫害，被关押五年半之久。在此期间，我没有得过什么病，精神没有垮下来。主要原因就是我牢记毛主席以上的这些话，从中得到了精神支柱。我还发奋读书，读了马恩全集、马恩通信集，列宁的主要著作，毛选四卷，还有鲁迅的若干著作。由此我也深感得益匪浅。作为一个担任过省委第一书记的人，我经受了受冤屈的考验，算是没有辜负毛主席的教诲吧。<br><br>1996年2月

## 🧠 深度思考与解读

## 1. 历史场域与战略意图

本章内容引自毛泽东在1956年9月中共八大预备会议上的讲话。这是一个极其关键的历史节点，其时局背景复杂而深远：

- **时局**：1956年2月，苏共二十大赫鲁晓夫秘密报告全盘否定斯大林，引发了国际共产主义运动的巨大震动和思想混乱。这不仅动摇了许多共产党人的信念，也暴露了社会主义国家内部权力斗争、个人崇拜和冤假错案的严重问题。中国共产党面临着如何评价斯大林、如何避免重蹈覆辙、如何保持党的团结和思想纯洁性的严峻考验。同时，国内大跃进的序幕即将拉开，党内对经济建设的路径、方法也存在不同认识，一些干部可能因过去的错误或路线斗争而心灰意冷。

- **靶子**：毛泽东此番讲话的“靶子”并非指向某个具体个人或派别，而是针对一种普遍存在的、可能动摇党组织根基的危险倾向：即在遭受政治打击、个人冤屈或路线批判后，产生的消极不满、一蹶不振，甚至对党和真理失去信心的情绪。他以自身20次受打击的经历为例，尤其提及“枪杆子出政权是错误的”、“山上没有马克思主义”等批判，这在当时是党内路线斗争的真实写照，也暗示了党内曾存在教条主义、脱离中国革命实际的错误倾向。

- **意图**：毛泽东的战略意图是多重的：
    1. **统一思想，增强党内凝聚力**：在苏共二十大引发的思想冲击下，通过强调党的自我纠错能力和最终会坚持真理的属性，稳定干部队伍，防止因个人遭遇不公而对党产生动摇。
    2. **教育干部，培养政治韧性**：教导干部如何正确对待个人逆境，将其转化为学习和提升的契机，而非沉沦。这对于即将到来的复杂政治运动和建设任务至关重要。
    3. **重申实事求是原则**：通过“白的说不黑，黑的说不白”强调客观真理的不可磨灭性，以及党最终会坚持 [[客观规律至上原则]]，为未来的路线斗争和政策调整奠定思想基础。
    4. **警示权力滥用**：虽然未直接批判，但提及“冤屈”和“平反”，也间接警示了党内可能存在的权力滥用和主观主义，强调了党最终会通过其 [[组织免疫系统激活机制]] 来纠正错误。

## 2. 核心方法论建模

面对上述危机与挑战，毛泽东提出了以下可复用的思维模型和工作法则：

### [[逆境驱动学习模型]]

**定义**：将个人或组织遭遇的挫折、不公甚至迫害，视为深化理论学习、联系实践、提升认知水平的强大驱动力的思维框架。

**操作步骤**：
1. **识别逆境**：当个人遭遇不公、被误解、被边缘化或遭受打击时，不沉溺于情绪，而是将其视为一个客观存在的“事件”。
2. **内化反思**：将外部的打击转化为向内求索的动力，反思自身在理论、实践或方法论上的不足。
3. **强制学习**：利用被“撤职”、“没事干”等空闲时间，系统性地、饥渴地阅读经典理论著作，弥补知识短板。
4. **联系实践**：将所学理论与个人经历、历史教训（如陈独秀、王明的错误）以及中国革命的具体实践相结合，进行深度思考和验证。
5. **升华认知**：通过理论与实践的结合，形成对事物更深刻、更全面的理解，从而找到解决问题、推动进步的正确道路。

### [[真理自信原则]]

**定义**：在复杂甚至被误解的环境中，对客观真理及其最终会显现的坚定信念，以此作为个人行动和组织决策的根本锚点。

**操作步骤**：
1. **坚守客观事实**：无论外部如何评判，坚信“白的说不黑，黑的说不白”，即客观事实和真理不会因主观意志或权力压制而改变。
2. **锚定核心信念**：相信组织（党）的本质是“掌握真理的党，是实事求是的党”，其最终目标是追求真理和人民的利益。
3. **时间检验**：认识到真理的显现和冤屈的平反可能需要漫长的时间（“十年二十年甚至更长的时间”），但最终会实现。
4. **超越个人得失**：将个人遭遇置于历史长河和组织大局中考量，不因一时一地的得失而动摇对真理和组织的根本信念。

### [[政治韧性培养法则]]

**定义**：通过持续的理论学习、实践检验和对核心原则的坚守，在面对政治打击、思想批判或个人冤屈时，保持精神不垮、信念不移的能力。

**操作步骤**：
1. **精神支柱构建**：在逆境中，主动寻找并强化精神支柱，如毛泽东的讲话、马列主义经典著作等，作为对抗消极情绪的内在力量。
2. **主动学习与思考**：将学习视为一种积极的应对策略，通过知识武装头脑，提升分析和解决问题的能力，从而增强自信。
3. **实践检验真理**：在学习的基础上，积极参与实践，用实践成果来验证和巩固自己的信念，形成正向循环。
4. **长期主义视角**：认识到政治斗争和个人成长都是一个长期过程，保持战略定力，不为短期挫折所困扰。

## 3. 哲学底层逻辑

这些方法论深刻体现了马克思主义的认识论和辩证法原理：

- **认识论：[[实践出真知]]**：毛泽东的亲身经历是这一原理的最好注脚。他因被“撤职，没事干，逼得我只好读书”，并将所读之书“联系中国革命的具体实践”，从血的教训中“寻求中国革命的正确道路”。这正是从感性认识到理性认识，再回到实践检验和指导实践的完整过程。个人遭受的“冤屈”本身是一种实践的挫折，但通过理论学习和对历史实践的深刻反思，反而深化了对真理的认识。这体现了认识的螺旋式上升，即挫折也是认识深化的重要环节。

- **辩证法：矛盾的转化与两点论**：
    - **矛盾的转化**：个人遭受的“冤屈”和“打击”看似是消极的、负面的矛盾，但毛泽东将其视为“有益的教育”，通过积极的应对（读书、寻求真理），将消极矛盾转化为积极的动力，实现了从“坏”到“好”的转化。这体现了矛盾双方在一定条件下可以相互转化的辩证思想。
    - **内因与外因**：虽然外部的打击是外因，但毛泽东强调的是个人如何通过内因（发奋读书、坚定信念）来应对和转化外因的影响。他指出“错误不在你，而在封你为机会主义的人”，这既是对外部错误的客观判断，更是强调了个人坚守真理的内在定力。
    - **[[客观规律至上原则]]**： “白的说不黑，黑的说不白”是唯物主义认识论的体现，强调客观真理不以人的主观意志为转移。无论权力如何扭曲，事实和规律最终会显现，这为在逆境中坚守信念提供了哲学支撑。

## 4. 现实场景迁移

历史总是押韵的。在现代职场、企业管理乃至个人成长中，我们依然能看到类似的“历史剧本”上演，尤其是在组织变革、权力斗争或绩效评估不公的背景下：

- **场景映射**：
    - **企业内部的“路线斗争”**：当企业进行战略转型或组织架构调整时，不同部门、不同派系之间可能出现对技术路线、市场策略、管理模式的激烈争论。一些坚持旧有模式或被视为“保守派”的员工可能会被边缘化、甚至被“撤职”，类似于毛泽东当年被批“狭隘经验主义”。
    - **职场政治与不公**：员工可能因派系斗争、领导偏见或信息不对称而遭受不公正的绩效评估、晋升受阻，甚至被“穿小鞋”。这与毛泽东所说的“受冤屈”异曲同工。
    - **创新与守旧的冲突**：在推动创新时，新思想、新方法往往会受到传统势力的阻挠和批判，创新者可能被贴上“激进”、“不切实际”的标签，甚至被排挤。

- **行动建议**：
    1. **将挫折视为“强制性学习项目”**：当个人在职场遭遇不公或职业瓶颈时，不要沉溺于抱怨，而应主动启动 [[逆境驱动学习模型]]。利用这段“空闲”时间，系统学习行业前沿知识、管理理论或新兴技能，将外部压力转化为自我提升的内在动力，为下一次机会做好准备。例如，被降职的管理者可以利用这段时间深入学习数据分析或AI应用，提升自身硬实力。
    2. **锚定核心价值观与 [[真理自信原则]]**：在组织内部出现混乱、价值观模糊或个人遭遇不公时，管理者和员工都应坚守企业或团队的核心价值观，以及自身职业道德的底线。相信基于事实和客观规律的决策最终会得到认可，不因短期利益或外部压力而放弃原则。例如，在面对不合理的KPI或潜规则时，坚持以客户价值和长期发展为导向的原则。
    3. **培养 [[政治韧性培养法则]]，做长期主义者**：认识到职业发展和组织变革是一个长期且充满波折的过程。在面对挫折和不公时，保持积极乐观的心态，通过持续学习和实践来增强自身实力，提升抗压能力。相信“白的说不黑”，你的真实能力和贡献最终会被看见。例如，即使项目被否决，也要从中吸取教训，持续优化方案，等待时机，而非一蹶不振。
