# 第160章 友谊，还是侵略？

> **归属**：第四卷 / 第三次国内革命战争时期
> **发表时间**：一九四九年八月三十日

> **摘要**：本章借毛泽东揭露美国‘友谊’面纱的犀利笔触，映射职场中如何识破‘画饼文化’与‘情感绑架’。核心在于透过温情脉脉的‘企业文化’表象，看清利益分配的本质，拒绝被‘成长’、‘情怀’等糖衣炮弹PUA，建立基于契约与价值交换的清醒职业观。

为了寻找侵略的根据，艾奇逊重复地说了一大堆“友谊”，加上一大堆“原则”。<br><br>艾奇逊说：“从我们历史很早的时期起，美国人民和政府就关心中国了。虽然距离遥远，背景又大不相同，把中美两国隔离开了，可是那些在宗教、慈善事业和文化方面团结中美两国人民的纽带，一直在加深着美国对中国的友谊，许多年来种种善意措施便是证据，例如用庚子赔款来教育中国学生，在第二次世界大战期间废除治外法权，以及战时和战后对中国的大规模援助等等。美国始终维持并且现在依然维持对华外交政策的各项基本原则，包括门户开放主义，尊重中国行政和领土的完整，以及反对任何外国控制中国等等，这是有案可稽的。”<br><br>艾奇逊当面撒谎，将侵略写成了“友谊”。<br><br>美帝国主义侵略中国的历史，自从一八四〇年帮助英国人进行鸦片战争<br><br>〔1〕<br><br>起，直到被中国人民轰出中国止，应当写一本简明扼要的教科书，教育中国的青年人。美国是最早强迫中国给予治外法权<br><br>〔2〕<br><br>的国家之一，这即是白皮书上提到的中美两国有史以来第一次签订的一八四四年的望厦条约<br><br>〔3〕<br><br>。就是在这个条约里，美国除了强迫中国接受五口通商等事而外，强迫中国接受美国人传教也是一条。美帝国主义比较其他帝国主义国家，在很长的时期内，更加注重精神侵略方面的活动，由宗教事业而推广到“慈善”事业和文化事业。据有人统计，美国教会、“慈善”机关在中国的投资，总额达四千一百九十万美元；在教会财产中，医药费占百分之十四点七，教育费占百分之三十八点二，宗教活动费占百分之四十七点一<br><br>〔4〕<br><br>。我国许多有名的学校如燕京、协和、汇文、圣约翰、金陵、东吴、之江、湘雅、华西、岭南等，都是美国人设立的<br><br>〔5〕<br><br>。司徒雷登就是从事这些事业出了名，因而做了驻华大使的。艾奇逊们心中有数，所谓“那些在宗教、慈善事业和文化方面团结中美两国人民的纽带，一直在加深着美国对中国的友谊”，是有来历的。从一八四四年订约时算起，美国在这些事业上处心积虑地经营了一百零五年，据说都是为了“加深友谊”。<br><br>参加八国联军打败中国<br><br>〔6〕<br><br>，迫出庚子赔款，又用之于“教育中国学生”，从事精神侵略，也算一项“友谊”的表示。<br><br>治外法权是“废除”了，强奸沈崇案的犯人回到美国，却被美国海军部宣布无罪释放<br><br>〔7〕<br><br>，也算一项“友谊”的表示。<br><br>“战时和战后的对华援助”，据白皮书说是四十五亿余美元，据我们统计是五十九亿一千四百余万美元，帮助蒋介石杀死几百万中国人，也算一项“友谊”的表示。<br><br>所有一百零九年（从一八四〇年英美合作的鸦片战争算起）美帝国主义给予中国的“友谊”，特别是最近数年帮助蒋介石杀死几百万中国人这一项伟大的“友谊”，都是为着一个目的，就是“始终维持并且现在依然维持对华外交政策的各项基本原则，包括门户开放主义，尊重中国行政和领土的完整，以及反对任何外国控制中国等等”。<br><br>杀死几百万中国人，不为别的，第一为了门户开放，第二为了尊重中国行政和领土的完整，第三为了反对任何外国控制中国。<br><br>现在，只有广州、台湾等处一小片地方的门户，还向艾奇逊们开放着，第一个神圣的原则在那里“依然维持”着。其余的地方，比如上海吧，解放以后本来是开放的，现在却被人用美国的军舰和军舰上所装的大炮，实行了一条很不神圣的原则：门户封锁。<br><br>现在，只有广州、台湾等处一小片地方的行政和领土，还算叨了艾奇逊第二个神圣原则的光，“依然维持”住了它们的“完整”。其余地方，一概倒运，行政和领土都是破碎得不像样子了。<br><br>现在，只有广州、台湾等处地方，叨了第三个神圣原则的光，把“任何外国的控制”，连同美国的控制也在内，都给艾奇逊们“反对”掉了，因此还被中国人控制着。其余的国土，说来要掉眼泪，一概完了，都给外国人控制住了，中国人统统当了奴隶。至于是什么外国呢，艾奇逊老爷行文至此，还没有来得及点出，下文自明，无须多问。<br><br>不干涉中国内政，是否也算一条原则呢，艾奇逊没有说，大概不能算吧。美国老爷的逻辑，就是这样。看完艾奇逊信件的全文，就可以证实这一项高明的逻辑。<br><br>注　　释<br><br>〔1〕<br><br>见本书第一卷《论反对日本帝国主义的策略》注〔35〕。<br><br>〔2〕<br><br>这里所说的“治外法权”，是指领事裁判权。见本书第二卷《中国革命和中国共产党》注〔22〕。<br><br>〔3〕<br><br>《望厦条约》是美国侵略中国的第一个不平等条约。一八四四年七月，美国利用中国在鸦片战争失败后的处境，强 ## 🧠 深度思考与解读

## 1. 历史场域与职场映射 (Context & Mapping)

**历史背景**：
1949年，美国发布《中美关系白皮书》，艾奇逊在信中大谈美国对中国的“友谊”（如庚子赔款办学、医疗援助），试图用“恩惠”来掩盖其支持蒋介石打内战、控制中国的实质。毛泽东针锋相对，一针见血地指出这些“慈善”和“文化”是**“精神侵略”**，是裹着糖衣的毒药。

**职场映射**：
这简直是当代互联网大厂**“企业文化洗脑”与“职场PUA”**的完美镜像。
- **艾奇逊的“友谊”** = **老板/HR的“画饼”**：
  - “我们是兄弟/家人” = “加班别谈钱”。
  - “这是给你的锻炼机会” = “脏活累活你全包，Headcount一个没有”。
  - “我们要拥抱变化” = “需求我想改就改，排期你必须守住”。
- **庚子赔款办学（精神侵略）** = **大厂光环下的“奋斗逼”文化**：通过输出一套看似高大上的价值观（如“福报论”），让你在潜意识里觉得“不996就是不努力”，从而自愿放弃休息权和议价权。
- **治外法权** = **业务方的免责金牌**：业务方（美国人）可以在系统里胡作非为（提伪需求），出了事（强奸沈崇案/系统崩溃）却由技术方（中国人）背锅或负责清理。

## 2. 执行层方法论建模 (Execution Methodology)

面对职场中的“艾奇逊”，核心员工不能做“待宰的羔羊”，而要学习毛泽东的**[[逻辑祛魅]]**战术。

### 2.1 拒绝“精神治外法权”
原文中，美国人享有“治外法权”，犯了法不由中国处置。在项目中，这对应**“权责不对等”**。
- **现象**：产品经理拍脑袋定需求，上线后无数据效果，不仅不背锅，还怪技术实现得慢。
- **行动法则**：**建立“引渡条约” (SLA与埋点)**。
  - 任何“友谊性”的紧急需求，必须有书面记录（Jira/邮件）。
  - 上线前必须定义成功指标（Success Metrics）。如果业务方说“先上了再说”，你要坚持“无指标不开发”。
  - **复盘机制**：如果需求上线后没流量，下次该业务方再提“紧急需求”，直接甩出历史数据拒绝，废除他的“治外法权”。

### 2.2 识别并拆解“糖衣炮弹”
毛泽东通过列举数据（杀了几百万中国人 vs 所谓的友谊），戳穿了谎言。你需要掌握**[[糖衣炮弹识别术]]**。
- **话术**：“这个模块很重要，交给你是因为信任你，虽然时间紧，但能证明你的能力。”
- **解码**：
  - “很重要” = 出了事是P0级事故。
  - “时间紧” = 资源不足，且不会补人。
  - “证明能力” = 做好了是本分，做坏了是背锅。
- **反制**：**“感谢信任，为了不辜负这份信任（糖衣），我需要X资源来保障质量（把炮弹打回去）。”** 不要直接拒绝，而是开出“对等条件”。

## 3. 哲学底层逻辑 (Philosophical Root)

### 3.1 现象与本质的辩证法
艾奇逊列举了“医院、学校、救济”等**现象**，试图证明“友谊”；毛泽东抓住了“侵犯主权、支持反动派”的**本质**。
- **职场启示**：不要被表面的福利（免费晚餐、健身房、期权大饼）迷惑。工作的本质是**价值交换**。如果一家公司给你很好的零食，却让你长期维护屎山代码且没有技术成长，这就是“精神侵略”——它在消耗你的职业寿命。

### 3.2 唯物主义的利益分析
毛泽东问：为什么美国要给钱？是为了“门户开放”，为了控制。
- **职场启示**：**谁受益，谁负责**。当跨部门协作中出现“友谊”时（比如兄弟部门求你帮忙跑个数据），先分析KPI归谁。如果KPI归他，锅归你，这种“友谊”必须断交。坚持**“亲兄弟，明算账”**的接口契约精神。

## 4. 现实场景迁移 (Pragmatic Application)

### 痛点：面对老板的“情怀绑架”无法拒绝
**场景**：老板说“我们要不惜一切代价拿下这个山头（项目），大家周末冲一下，年底我有数。”

**行动指南**：
1.  **数据化防御（Data Defense）**：
    不要情绪化反驳。学习毛泽东列举“四十五亿 vs 五十九亿”的数据对比法。
    *话术*：“老板，为了拿下山头，我盘点了目前的弹药（人力）。如果全员周末冲刺，根据历史Bug率，下周系统的稳定性风险会增加30%。我们可以冲，但建议把非核心功能的‘门户’先关掉（砍需求），集中兵力。”

2.  **建立“反精神侵略”防火墙**：
    将工作与个人价值剥离。公司不是家，同事不是家人。
    - **文档即法律**：所有口头承诺的“友谊”（年底加薪、调休），尽量落实到文字或邮件。
    - **技术根据地**：不管公司怎么画饼，你必须守住自己的**[[技术根据地]]**（核心技能栈）。如果公司的业务让你沦为纯粹的CRUD工具人，哪怕饼再大，也要准备跑路，因为你的“领土完整”（职业竞争力）正在被侵蚀。

3.  **避坑指南**：
    - **不要做“义和团”**：毛泽东反对美帝，但没说要烧教堂、杀教士（盲目排外）。在职场中，不要因为看穿了老板的画饼就变成“杠精”或“刺头”。要**“有理、有利、有节”**。表面上维持“友谊”（配合工作），实际上寸土不让（守住边界和排期）。
    - **警惕“买办思维”**：不要为了讨好上级，就出卖团队兄弟的利益（例如Tech Lead盲目承诺工期，压榨组员）。这最终会导致众叛亲离，失去你的**核心基本盘**。
