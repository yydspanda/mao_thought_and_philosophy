# 第383章 中美关系会一点点好起来

> **归属**：第七卷（静火版） / 文化大革命时期
> **发表时间**：一九七五年十二月二日

> **摘要**：在1975年与美国总统福特的会谈中，毛泽东通过战略性示弱（放空炮）来探测和倒逼美国拿出对抗苏联扩张的实际行动（实心炮），并以“抓主要矛盾”的哲学思想为核心，主导了一场全球地缘政治的“君子协定”，旨在构建最广泛的国际反霸统一战线。

福特：主席先生，我们今天早上有一次很不错的讨论。<br><br>毛泽东：你们讨论什么？<br><br>福特：我们讨论了我们同苏联之间的问题，还有我们在全球范围内的对策，以及我的国家和你的国家有必要合作，去实现那些对我们两国都有利的目标。<br><br>毛泽东：我们没有这种本事，我们只能放放空炮。<br><br>福特：我不相信，主席先生。<br><br>毛泽东：如果说到骂人，这种本事我们倒是有一点。<br><br>福特：我们也能。<br><br>毛泽东：你们也能？那么我们应该达成一项协议。<br><br>福特：我们还可以使用武力，对付那些制造麻烦的国家。<br><br>毛泽东：这个不坏。那么我们可以达成另外一项协议。<br><br>福特：今天早上，我们讨论的时候，有很明确的讨论目标。<br><br>毛泽东：除了苏联帝国主义者，不可能是别人了。<br><br>福特：主席先生，今天早上，我们使用了一些很强硬的语言。<br><br>毛泽东（指着邓小平〔1）〕：你们批评他了？<br><br>福特：我们强烈批评的是另外一个国家。<br><br>毛泽东：那个在北面的国家。<br><br>福特：对。<br><br>毛泽东（对驻美大使黄镇）：黄镇<br><br>〔2〕<br><br>同志，事情怎么样了？你还回美国吗？<br><br>黄镇：我听从主席的指示。<br><br>毛泽东：总统先生，你们要他吗？<br><br>福特：我们当然要他回去。我们的关系好极了。大使先生回美国是很重要的，就像（驻华大使）布什<br><br>〔3〕<br><br>先生要是留在北京一样。<br><br>毛泽东（对布什）：你留在北京吗？<br><br>布什：只有几天了。<br><br>毛泽东：你升职了。<br><br>福特：对，他升职了。我们将在一个月里，提名继任者。<br><br>毛泽东：我们不想让他走。<br><br>福特：他是一个出色的人，所以我要他回美国。但我们会找一个同样好的人来接替他。<br><br>毛泽东：那样挺好。我觉得，黄镇同志返回美国更有利。<br><br>黄镇：我将坚决执行主席的指示。我确实想回国，因为我驻外太久了。但是我将按照主席说的做。<br><br>毛泽东：你应该再在那里待一到二年。<br><br>黄镇：好的，我肯定会回去，坚决执行主席的指示。<br><br>毛泽东：有一些年轻人批评他（指黄镇）。这两个年轻人（指在场的两个翻译唐闻生、王海容）对乔老爷<br><br>〔4〕<br><br>也有一些批评。不能忽视这些人啊，否则你会在他们身上吃苦头的。这是一场内战。现在外面有很多大字报出来。你不妨去清华和北大看看。<br><br>福特：我不太明白这些东西。主席先生，我希望，你让大使回美国再待两年，意味着我们还能继续发展我们两国之间的良好关系。<br><br>毛泽东：对，对，我们之间的关系还要继续发展。我觉得，现在我们两国之间没有什么大问题，你的国家和我的国家。今年、明年、后年，可能都不会有大问题发生在我们两国之间。以后的情况可能会变得更好一点吧。<br><br>福特：主席先生，我认为我们不得不在国际事务上加强合作，应对来自诸如苏联那样国家的挑战。<br><br>毛泽东：对，我们对苏联一点信心也没有。邓小平同志也不喜欢苏联。<br><br>福特：我们也是这样。他们在全世界扩张，掠夺领土和经济资源。我们必须面对他们的挑战。<br><br>毛泽东：好。我们也要面对挑战。<br><br>福特：我们希望从明年开始，我们的双边关系可以得到改善。我们认为，现在正是我们两国关系取得实质性进展的时候。<br><br>毛泽东：你是说我们两国之间？<br><br>福特：对。<br><br>毛泽东：那样很好。<br><br>福特：主席先生，如果你的国家和我的国家一起合作，应对来自苏联的挑战，那么美国和中华人民共和国外交关系正常化，就会在我的国家得到支持。<br><br>毛泽东：很好。不过，我们在这里只是说说而已，苏联到底怎么行动，还需要观察。<br><br>福特：主席先生，我们必须让苏联对美国和中华人民共和国的合作留下深刻印象，言辞是不起作用的，必须有实际行动。我们将不断对他们施加压力。我希望，来自您的压力，和我们的压力一样强大。<br><br>毛泽东：我们就是放放空炮，骂骂人而已。<br><br>福特：主席先生，我们将要做的就不止与此了，过去也是如此。美国人民希望他们的总统坚强有力。我们将来就会这样行动。而不是只说不做、放空炮。<br><br>毛泽东：你们有实心炮？<br><br>福特：对，我们将一直准备火药，如果他们来挑战我们，那么我们的火药就不会闲着。<br><br>毛泽东：很好，那样也不坏。不过，现在你们是和平共处。<br><br>福特：但是，这不意味着我们会对任何 ## 1. 历史场域与战略意图 (Context & Intent)

这次谈话发生于1975年12月，这是一个极其微妙的历史节点，理解其背景是解读一切的前提。

- **时局**：
  1.  **全球“苏攻美守”**：美国刚输掉越南战争（1975年4月西贡陷落），国内因“水门事件”元气大伤，战略上处于收缩和迷茫期。苏联则处于勃列日涅夫治下的鼎盛期，全球扩张势头正盛，尤其是在非洲（安哥拉内战）和欧洲，形成了咄咄逼逼人的“苏攻美守”态势。
  2.  **中国内部的“风暴前夜”**：毛泽东已入暮年，健康状况不佳。邓小平复出后大刀阔斧进行整顿，与“四人帮”的矛盾日益尖锐。外交路线是两派斗争的焦点之一，激进派反对与美帝“勾结”。此次会谈既是外交，也是对内的政治宣示。

- **靶子**：
  1.  **外部靶子**：明确无疑是**苏联霸权主义**。整篇谈话的核心议题就是如何共同应对“那个在北面的国家”的全球扩张。
  2.  **内部靶子**：毛泽东提及“年轻人批评”外交官黄镇、乔冠华，以及清华北大的大字报，是隐晦地指向党内以“四人帮”为首的激进派。他们干扰和破坏了中国基于“一条线、一大片”反霸战略的务实外交布局。

- **意图**：
  1.  **战略再确认**：尼克松破冰、中美关系正常化的大门已经打开，但福特是新总统。毛泽东的首要意图是**“战略验真”**——探明美国新政府在“联华抗苏”这一核心战略上的决心和意志是否动摇。
  2.  **倒逼美国拿出行动**：毛泽东敏锐地察觉到美国战败后的“心气”不足。他担心美国的“缓和政策”会变成对苏联的绥靖。因此，他要通过言语刺激，逼迫福特政府从“口头反苏”走向“行动反苏”。
  3.  **构建全球反霸“指挥图”**：谈话遍及欧洲、中东、南亚、非洲，毛泽东如同一位全球战略总棋手，与美方逐一“对表”，确认在各个关键地缘节点上的共同利益和协同策略，意图构建一个全球范围的反苏统一战线。

这是一种典型的以退为进、虚实结合的谈判模型，旨在探测对方底牌并锁定其承诺。

- **操作步骤**：
  1.  **战略性示弱（放空炮）**：开篇即表明自身能力有限。当福特提出中美合作对抗苏联时，毛泽东立刻回答：“我们没有这种本事，我们只能放放空炮。” 这句话看似谦虚，实则将“行动”的皮球踢给了实力更强的美国。
  2.  **激发对方亮牌（引实炮）**：这种示弱会迫使急于寻求合作的对方，必须展示自己的实力和决心来增强你的信心。福特马上反驳：“我不相信”，并强调“我们还可以使用武力”。
  3.  **确认承诺（验实炮）**：毛泽东不轻易相信，而是进一步追问：“你们有实心炮？” 这句点睛之笔，将模糊的“决心”聚焦于“实力和行动”上，迫使福特做出更具体的承诺：“我们的火药就不会闲着。”
  4.  **达成默契（锁定目标）**：通过反复的“空炮”与“实炮”的对话，双方最终就“必须有实际行动”达成共识，完成了从意向到行动承诺的关键一步。

### 模型二：[[全球压力点共识模型]] (Global Pressure Point Consensus Model)

此模型超越了双边关系，旨在通过对全球棋局的共同复盘，建立一个广泛的、非正式的战略同盟。

- **操作步骤**：
  1.  **确立共同威胁**：首先锁定核心敌人——苏联，并就其“扩张主义”的性质达成一致。
  2.  **全球巡视与诊断**：谈话如同一场战略地图推演，从欧洲（葡萄牙、西班牙、南斯拉夫）到亚洲（日本、南亚次大陆），再到非洲（安哥拉），逐一分析苏联在各地的渗透和挑战。
  3.  **划分势力范围与责任区**：在讨论中，自然形成了分工。例如，美国负责稳固北约南翼（西班牙、希腊），利用其在中东的影响力；中国则可以尝试对莫桑比克施加影响，并通过扎伊尔支援安哥拉的反苏派别。
  4.  **形成“君子协定”**：在关键问题上达成行动默契。毛泽东明确提出：“你从东边施加压力，我从西边施加压力。” “可以。这是君子协定。” 这种非正式协议比正式条约更灵活，也更符合当时中美关系的性质。

## 3. 哲学底层逻辑 (Philosophical Root)

这些高超的战略方法论，根植于毛泽东深刻的马克思主义哲学思想。

- **矛盾论是总开关**：整个谈话的灵魂是[[抓主要矛盾]]。在毛泽东看来，当时世界的主要矛盾是美苏争霸，特别是苏联的社会帝国主义扩张。中美之间意识形态的矛盾是次要的。因此，他可以坦然地说“我们之间没有任何矛盾”，并在此基础上建立最广泛的统一战线。解决主要矛盾，是所有策略的出发点和归宿。

- **辩证法是工具箱**：
  1.  **两点论与重点论的统一**：毛泽东清楚中美有制度差异（两点论），但他牢牢抓住“反霸”这个共同利益（重点论），使得谈话始终围绕核心目标展开，不被次要分歧带偏。
  2.  **对立统一**：中美这两个曾经的战场死敌，在苏联这个更强大的威胁面前，形成了“对立面的统一”。这种统一是暂时的、有条件的，但又是现实的、必要的。毛泽东将这种哲学思辨纯熟地运用于国际政治实践。

- **认识论的实践观**：毛泽东对“言辞”抱有天然的警惕。他反复强调“言辞是不起作用的，必须有实际行动”，“苏联到底怎么行动，还需要观察”。这体现了他“实践是检验真理的唯一标准”的认识论。他不听美国怎么说，而要看美国怎么做，是否真的准备了“火药”和“实心炮”。

## 4. 现实场景迁移 (Pragmatic Application)

这段看似遥远的大国博弈，对现代企业管理和个人发展有着惊人的启示。

- **场景映射**：
  - **主要矛盾**：在企业中，这个“苏联”可能是那个占据80%市场份额的巨头，也可能是颠覆行业的技术浪潮（如AI），或者是僵化的内部流程。很多企业失败，不是因为不够努力，而是没有[[抓主要矛盾]]，在次要问题上空耗资源。
  - **战略空炮**：在跨部门合作中，A部门经常口头承诺“全力支持”，但资源迟迟不到位。这就是只有“空炮”没有“实心炮”。

- **行动建议**：
  1.  **用“空炮探测法”获取真实资源**：当你需要关键部门或领导支持一个创新项目时，不要仅仅去汇报方案多好，可以尝试[[战略空炮探测模型]]。你可以说：“老板/X总，这个方案的创意和初步框架我们团队能搞定（放空炮），但要真正落地产生市场冲击力，必须依赖您亲自协调市场部的核心渠道资源（引实炮）。否则，它最多是个漂亮的PPT。” 这种方式能有效将口头支持转化为实实在在的资源承诺。

  2.  **召开“全球压力点”式的战略会议**：面对强大竞争对手，CEO可以召集核心团队，模仿毛泽东与福特的谈话模式。在白板上画出市场地图，逐一分析对手在“产品A线”、“华南市场”、“线上渠道”等各个“压力点”的优势和弱点。然后明确“我们销售部从渠道反击（西线），产品部用迭代版本牵制（东线）”，形成一个分工明确、协同作战的“君子协定”。

  3.  **永远聚焦于解决你的“主要矛盾”**：作为个人，你的“苏联”可能是职业发展的最大瓶颈（如技能短板、学历限制）。你要把80%的精力投入到解决这个主要矛盾上。与同事的次要矛盾、暂时的项目得失，都应服务于这个大局。分清主次，才能避免在琐事中沉沦，实现战略性突破。
