# 第29章 关于读书的方法

> **摘要**：在1958年大跃进初期面临思想混乱和政策执行偏差之际，毛泽东通过重新评价曹操、纣王、秦始皇等历史人物，提出“联系”与“评论员”的读书方法，旨在纠正党内教条主义和分散主义，强调辩证分析历史、独立思考，并融合集中统一领导与群众路线，以培养更具批判性和实践性的领导干部。

1958年11月20日上午，毛主席召集柯庆施、李井泉、王任重和我四个人到他在武汉东湖畔的住所开座谈会。原来我们都认为主席要座谈的是预定在明天，即11月21日开始的中央工作会议和相继召开六中全会的问题。谁也没有料到，当主席和蔼可亲地招呼我们在客厅坐定之后，一开头却说：“今天找你们来谈谈陈寿的《三国志》。”<br><br>他习惯地拿起一根烟，一折两截，把一截放在桌上，一截插进烟嘴，点燃后轻轻吸了一口，然后问大家：“你们都看过这部书吗？”柯庆施首先回答：“看过，还看过《三国演义》呢。”毛主席侧身坐在座椅上向我们环顾了一下说道：“不过，《三国演义》是小说，《三国志》是史书，二者不可等同视之。若说生动形象，当然要推演义；若论真实性，就是说更接近历史真实，罗贯中的《三国演义》就不如陈寿的《三国志》啰！比如：旧戏里诸葛亮是须生，而周瑜是小生，显然诸葛亮比周瑜年纪大。这可能是来源于演义，然而《三国志》上记载周瑜死时36岁，那时诸葛亮才30岁，即比周瑜小6岁。”这时李井泉插话说：“书中那么多人物、事物，看过就忘了，不如主席记得那么清楚。”主席说，为什么记不清楚呢？要想记得清楚，看书就得讲究方法。读书的方法无非两条，第一条要学会用联系的方法看书中的人物、事件；第二条要学会当评论员。<br><br>在这次座谈会10天之前，即第一次郑州会议上，毛主席说：“把商纣王、秦始皇、曹操看作坏人是错误的。”在这次座谈会上，他详细地谈了自己对这三位历史人物的评价。<br><br>谈到曹操，毛主席说，你们读《三国演义》和《三国志》注意了没有，这两本书对曹操的评价是不同的。座中有的同志说，一个是贬，一个是褒。主席说，是的。《三国演义》是把曹操看作奸臣来描写的；而《三国志》是把曹操看作历史的正面人物来叙述的，而且说曹操是天下大乱时期出现的“非常之人”，“超世之杰”。可是因为《三国演义》又通俗又生动，所以看的人多，加上旧戏上演三国戏都是按《三国演义》为蓝本编造的。所以曹操在旧戏舞台上就是一个白脸奸臣。这一点可以说在我国是妇孺皆知的。毛主席说到这里，愤愤不平地说，现在我们要给曹操翻案。我们党是讲真理的党，凡是错案、冤案，十年、二十年要翻，一千年、二千年也要翻。他实事求是地评价曹操说：“曹操统一北方，创立魏国，抑制豪强，实行屯田，兴修水利，发展生产，使遭受大破坏的社会开始稳定和发展，是有功的。说曹操是奸臣，那是封建正统观念制造的冤案，这个冤案要翻。”<br><br>谈到纣王，毛主席说，史书上把纣王描写得像一个青面獠牙、十恶不赦的坏人。太过分了。连孟夫子也为他抱不平地说“尽信书则不如无书”，“桀纣之恶未有如此之甚也，是以君子恶居下流，天下之恶均归焉。”纣王宠妲己、剖比干心，这两件坏事，使他得到了一个大暴君的恶名，于是天下之恶就都归到纣王头上了。似乎他什么好事都没有做。其实纣王这个人聪明善辩,能武能文。他打起仗来是很有英雄气概的。商朝晚期，江淮之间的夷人强盛起来，威胁商朝，纣王的父辈曾几次对东夷用兵，得了些胜利，但没有打退东夷向商朝的扩张和侵犯。纣王当政后亲率大军东征夷人。打了一场空前的大胜仗，俘虏了“亿兆夷人”，由此打退了东夷的扩张，保卫了商朝在东南方的安全。而且纣王尚武重文，他对东南的经营，使中原文化逐渐发展到了东南，这对我国历史是有贡献的。毛主席说，商朝就是做生意的意思，它标志着商朝已开始有了商品交易，到纣王时己成为当时最富强的、文化最高的奴隶制国家。<br><br>毛主席说，纣王亡国的教训，一是对周文王、周武王励精图治、吸收商人文化，促进周族社会的发展，积极准备推翻商朝的统治，完全丧失了警惕性。根本不听商朝大贵族微子、箕子和王子比干的反复进谏。结果，比干被杀，箕子被囚为奴，微子见势不妙逃走了。纣王陷入了众叛亲离的绝境。一是纣王不会做俘虏工作，把大量战俘集中于京城附近；而周人在纣王的战俘中却做了内线工作。当周武王率领的大军距商都七十里的地方，商王宫廷才得到消息，赶忙停止为欢庆胜利而举行的歌舞和酒宴，开始讨论应敌的对策。这时商朝的主力军还远在东南战场，一时征调不回来，纣王只得把大量东夷战俘武装起来开向前线。结果，战俘纷纷起义，掉转矛头，配合周军进攻商纣。纣王见大势已去，登鹿台自焚而死。“纣克东夷而殒其身”，这就是商朝的结局，纣王的悲剧。<br><br>谈到秦始皇，毛主席说，人们从书中得知，秦始皇有焚书坑儒的恶行，因此把他看作是大暴君、大坏人。焚书坑儒当然是坏事，它把蓬蓬勃勃发展起来的百家争鸣的生动局面给挫折了。但我们对什么事都应当有分析，秦始皇并不是不问什么书都焚，也不是不问什么儒都坑。他焚的是“以古非今”的书，坑的是孟子一派的儒，其实只有460人。孟子主张“法先王”，所以孟子一派的书是“以古非今”的。而荀子一派则相反，主张“法后王”，推行法家一派的学说。秦始皇是主张“法后王”，反对“法先王”的。所以他并不坑荀子一派的儒，也不焚荀子一派的书。秦始皇“以古非今者族”的主张值得赞赏，当然我并不赞成秦始皇的滥杀人。当时，要由奴隶制国家转变为封建制国家，不实行专政是不行的。但对孟子一派采取焚书坑儒的办法，太过火了。政治上要实行专政，文化上要提倡百家争鸣、百花齐放，我们现在就是这样。这一条秦始皇是办不到的。<br><br>说秦始皇没有做过一件好事，太武断了。秦始皇第一个统一了中国，统一了原来各国的度量衡，车同轨，书同文字，变分封制为郡县制。这些事关中华民族兴盛的大事，能说不是好事吗？秦始皇还在陕西关中开凿了有名的郑国渠，长三百余里，可灌溉农田四万余顷，直接于生产有益，于人民有益。秦国也因此富强起来，终于把六国吞并了。能说这不是好事吗？<br><br>秦始皇是一个大独裁者，有些历史学家这样说不是没有道理的。毛主席认为，对于这一点也应该有分析。秦始皇有独裁的一面，也有高度集中统一领导的一面，二者有区别又有联系。后者在秦始皇吞并六国、统一中国的过程中，是他取得成功的一个积极因素。秦始皇当全局的情况看准之后，他善于调动各方面的力量集中到主攻方向上来，而且在实行主攻任务的时候，有很大的决心，很大的气魄，很顽强的毅力，敢于力排众议，不听那一套动摇军心的话。<br><br>在1959年4月上海会议上，毛主席针对当时工作中存在的分散主义、本位主义、有令不行、有禁不止的情况，提出：要学习和借鉴秦始皇善于集中力量于主攻方向，同时要学会走群众路线。也就是把集中统一的领导和群众路线统一起来。毛主席说，不利于群众的事情就是不利于国家，没有什么有利于群众而不利于国家的事。秦始皇并不是没有过错，给秦始皇翻案，要看到他还有重大的过错。历史上的秦始皇搞专制独裁，同群众路线是根本对立的。现在我们需要的是走群众路线的“秦始皇”。一方面领导上不要给人乱戴帽子，允许保留意见，言者无罪，闻者足戒；一方面领导要以身作则，提倡作自我批评。这样两个方面结合起来，经过不同意见的争论，在这个基础上集中起来，这是革命的“秦始皇”，集中统一就有了群众基础。<br><br>为什么秦朝在秦始皇死后，不到三年就灭亡了？古来许多政治家、历史家做过多方面的研究。毛主席对汉朝贾谊的《过秦论》最为赞赏。他说，贾谊是政治家、历史家，他写的《过秦论》是以人民力量和人民的向背为基本立足点，来观察国家兴衰、帝王成败的。所以，《过秦论》最后的一句话概括秦朝速亡的原因是：“仁义不施，攻守之势异也。”用现在的话说，就是对人民施行暴政，丧失了民心。这是值得后世汲取的深刻教训。<br><br>毛主席对三个有争议的历史人物的评说，具体生动地体现了他提倡读历史书要用“联系”和当“评论员”的两条方法。所谓“联系”，按照辩证法,各种事物都有对立的两个侧面，“联系”就是对立的两个侧面的联系，而不是什么无缘无故的联系。如前所述，毛主席针对工作中存在着的分散主义，要求高级领导干部像秦始皇那样善于调动各方面的力量集中到主攻方向上来，以加强集中统一的领导。而集中统一的领导和分散主义在一个领导体系内是对立的两个侧面，它们在一定条件下，能够相互转化。这就是毛主席用“联系”的方法读史用史的例证之一。毛主席对三位历史人物的评论，说明他当“评论员”站得高，看得远，不囿于已有的历史结论，而有自己独立思考的独到见解。<br><br>总之，毛主席提倡的“联系”和当“评论员”的读书方法，二者是相辅相成不可分割的。只有“联系”才能拓宽视野，有所比较，学以致用；只有当“评论员”才能有分析，有鉴别，有结论。离开了“评论”的联系，就缺乏灵魂；离开了“联系”的评论就没有基础。<br><br>1993年9月<br><br>附件<br><br>昆明大观楼楹联<br><br>五百里滇池，奔来眼底。披襟岸帻，喜茫茫空阔无边。看，东骧神骏，西翥灵仪，北走蜿蜒，南翔缟素；高人韵士，何妨选胜登临，趁蟹屿螺洲，梳裹就风鬟云鬓，更苹天苇地，点缀些翠羽丹霞。莫辜负四周香稻，万顷晴沙，九夏芙蓉，三春杨柳。<br><br>数千年往事，注到心头。把酒凌虚，叹滚滚英雄谁在。想，汉习楼船，唐标铁柱，宋挥玉斧，元跨革囊；伟烈丰功，费尽移山心力，尽珠帘画栋，卷不及暮雨朝云；便断碣残碑，都付与苍烟落照。只赢得几杵疏，半江渔火，两行秋雁，一枕清霜。

## 🧠 深度思考与解读

## 1. 历史场域与战略意图 (Context & Intent)

1958年11月，正值中国“大跃进”运动如火如荼之际。彼时，全国上下沉浸在“一天等于二十年”的狂热氛围中，但随之而来的浮夸风、共产风、瞎指挥等问题已开始显现，并导致了严重的资源浪费和生产秩序混乱。尽管高层对这些问题已有初步察觉（如第一次郑州会议），但党内普遍存在着一种思想上的僵化和教条主义倾向，即对既定政策缺乏批判性反思，对历史人物和事件的评价也往往停留在简单化、脸谱化的层面。

毛泽东此次与柯庆施等地方大员的座谈，表面上是谈《三国志》，实则“醉翁之意不在酒”。他感知到的“风起于青萍之末”的风险，正是这种**思想上的僵化和认识论上的偏差**。他试图解决的党内矛盾包括：

1.  **教条主义与经验主义的盛行**：干部们习惯于照搬书本或上级指示，缺乏独立思考和辩证分析的能力，尤其是在面对“大跃进”中出现的新问题时，往往难以从复杂现象中把握本质。
2.  **分散主义与本位主义的抬头**：在政策执行层面，各地可能出现“有令不行，有禁不止”的现象，导致中央政策难以有效落地，或在地方执行中走样变形。
3.  **脱离实际的官僚主义倾向**：对历史人物的简单褒贬，反映出一种脱离具体历史语境、缺乏实事求是精神的思维方式，这种思维方式也容易导致在现实工作中脱离群众、脱离实际。

毛泽东发表此文的战略意图是：

*   **统一思想，纠正认识论偏差**：通过重新评价历史人物，教导干部们运用辩证唯物主义和历史唯物主义的观点分析问题，破除僵化的历史观和教条主义，为后续的政策调整和纠偏工作奠定思想基础。
*   **提升领导干部批判性思维与实践能力**：鼓励干部们“当评论员”，敢于质疑既有结论，独立思考，从而在复杂局面中做出正确判断。
*   **强化党的集中统一领导与群众路线的结合**：借鉴秦始皇集中力量的优点，同时强调必须与群众路线相结合，避免重蹈秦朝因“仁义不施”而速亡的覆辙，为“大跃进”的健康发展指明方向。

## 2. 核心方法论建模 (Methodology Modeling)

针对上述危机，毛泽东提出了以下破局方法，可提炼为可复用的思维模型：

### 2.1 [[辩证联系分析法]]

**操作步骤：**
1.  **超越表象，深入本质**：不满足于对事物或人物的表面印象或流行评价（如《三国演义》对曹操的脸谱化），而是要深入挖掘其行为背后的历史语境、动机和客观影响。
2.  **全面考察，两点论**：认识到任何事物都具有对立的两个侧面（如曹操的“奸臣”与“有功”，秦始皇的“独裁”与“集中统一领导”）。避免片面化、绝对化的评价，力求全面客观。
3.  **动态发展，相互转化**：理解事物在特定条件下会发生转化（如纣王因“克东夷”而强大，却因对俘虏工作不力而“殒其身”）。认识到矛盾的转化是推动事物发展的内在动力。
4.  **联系实际，学以致用**：将历史经验与当前面临的实际问题相联系，从历史中汲取智慧，指导现实工作（如从秦始皇的集中统一领导中学习如何解决“分散主义”）。

### 2.2 [[独立批判性思维模型]]

**操作步骤：**
1.  **质疑权威，不尽信书**：面对既定的历史结论或流行观点，保持怀疑精神，不盲目接受。如毛泽东所言“尽信书则不如无书”。
2.  **搜集多方信息，交叉验证**：不局限于单一来源（如《三国演义》），而是要查阅更可靠的史料（如《三国志》），进行比较分析，还原历史真实。
3.  **形成独立判断，敢于“翻案”**：在充分分析和论证的基础上，形成自己的独到见解，并敢于挑战甚至推翻错误的传统结论，坚持真理。
4.  **以史为鉴，评论当下**：将历史人物和事件作为“评论员”的素材，通过评论历史来反思和指导现实，提升决策的洞察力。

### 2.3 [[历史重估与纠偏机制]]

**操作步骤：**
1.  **识别“冤案”或“错案”**：主动发现那些因历史局限、意识形态偏见或流行叙事而形成的、与客观事实不符的评价或结论。
2.  **启动“翻案”程序**：组织力量进行深入的史料考证和多维度分析，还原事件真相，纠正不公正的评价。
3.  **坚持真理，不惧阻力**：明确“我们党是讲真理的党”，即使是千年旧案，只要是错的，就必须纠正，这体现了党实事求是的根本原则。
4.  **以纠偏促进步**：通过纠正历史错误，不仅能恢复历史的本来面目，更能从中汲取深刻教训，避免在现实中重蹈覆辙，提升组织的自我修正能力。

### 2.4 [[集中统一与群众路线融合模型]]

**操作步骤：**
1.  **明确主攻方向，集中力量**：学习秦始皇善于“调动各方面的力量集中到主攻方向上来”，确保战略目标清晰，资源配置高效。
2.  **警惕独裁，避免脱离群众**：认识到秦始皇因“仁义不施，攻守之势异也”而速亡的教训，避免领导者脱离群众、施行暴政。
3.  **构建心理安全，鼓励言路**：领导者要“不要给人乱戴帽子，允许保留意见，言者无罪，闻者足戒”，营造一个鼓励讲真话、敢于批评的氛围。
4.  **领导垂范，自我批评**：领导者要“以身作则，提倡作自我批评”，通过榜样作用带动组织内部的批评与自我批评。
5.  **辩论中集中，群众中统一**：通过不同意见的争论，在充分吸收群众智慧的基础上形成集中统一的领导，使集中统一具有深厚的群众基础，成为“革命的‘秦始皇’”。

## 3. 哲学底层逻辑 (Philosophical Root)

这些方法论深刻体现了马克思主义哲学中的**矛盾论、认识论和辩证法**原理。

*   **矛盾论**：毛泽东对历史人物的评价，始终贯穿了对立统一的观点。他指出曹操的“奸臣”与“有功”、秦始皇的“独裁”与“高度集中统一领导”是事物内部的矛盾统一体。纣王“克东夷而殒其身”则揭示了主要矛盾与次要矛盾在特定条件下的转化，以及量变到质变的辩证关系。他强调“联系”的方法，正是要把握事物内部和事物之间的矛盾关系，从而全面、深刻地认识事物。

*   **认识论**：毛泽东的“尽信书则不如无书”和“当评论员”的思想，是对唯心主义和经验主义认识论的批判，强调了实践是检验真理的唯一标准，以及认识的能动性和批判性。他鼓励干部们通过深入分析史料、独立思考，从感性认识上升到理性认识，再通过实践检验和发展认识，体现了从实践到认识、再从认识到实践的辩证统一的认识路线。

*   **辩证法**：
    *   **两点论与重点论**：在评价历史人物时，毛泽东既看到了他们的缺点和错误（两点论），又强调了他们的主要贡献和历史进步作用（重点论），尤其是在纠正传统偏见时，更侧重于揭示其被忽视的积极面。这避免了形而上学的片面性。
    *   **具体问题具体分析**：他强调要结合具体的历史语境来评价人物，而不是用抽象的道德标准一概而论，这是马克思主义活的灵魂。
    *   **对立统一规律**：[[集中统一与群众路线融合模型]]是典型的对立统一思想。集中统一是效率和方向的保证，群众路线是活力和正确性的源泉，二者看似对立，实则相互依存、相互促进，在革命的实践中必须有机结合，才能实现真正的强大。

## 4. 现实场景迁移 (Pragmatic Application)

历史总是押韵的。在现代职场和企业管理中，我们依然能看到类似的“历史重演”：

*   **“修正主义”与“教条主义”现象**：
    *   **教条主义**：企业盲目追捧“最佳实践”或成功案例，不结合自身实际情况进行批判性分析和适应性调整，导致“水土不服”。例如，照搬硅谷的扁平化管理，却忽视了传统行业的层级需求；或固守过去的成功经验，对市场变化视而不见。
    *   **“冤案”与“脸谱化”**：对企业内部的失败项目、离职员工或竞争对手，往往存在简单粗暴的负面评价，缺乏深入分析其失败的复杂原因或对方的合理之处，导致无法从错误中真正学习。
    *   **领导层脱离一线**：高层管理者沉浸在数据报表和会议中，缺乏深入基层、倾听一线员工声音的“群众路线”，导致决策脱离实际，难以落地。
    *   **信息茧房与霸王困境**：领导者因害怕异议而压制不同声音，导致信息失真，最终陷入“[[霸王困境 (The Tyrant's Dilemma)]]”，决策失误，信任流失。

*   **行动建议**：

1.  **建立“历史复盘与反思机制”**：企业应定期对重大项目、战略决策进行深度复盘，不仅关注结果，更要分析过程中的决策逻辑、内外因素和潜在矛盾。鼓励团队运用[[辩证联系分析法]]，从成功和失败中全面挖掘经验教训，甚至对过去的“失败案例”进行[[历史重估与纠偏机制]]，发现其被忽视的积极意义或深层原因，避免简单归因和“甩锅”。

2.  **培养“批判性思维与心理安全文化”**：管理者应积极倡导[[独立批判性思维模型]]，鼓励员工对现有流程、产品甚至领导决策提出建设性意见。通过建立“[[心理安全构建法]]”机制（如匿名反馈、开放式讨论、领导者自我批评），确保“言者无罪，闻者足戒”，让不同声音成为决策优化的宝贵资源，而非威胁。

3.  **实践“双向赋能的领导力模型”**：在战略制定和执行中，高层领导要像“革命的秦始皇”一样，既要保持战略方向的[[集中统一与群众路线融合模型]]，确保资源聚焦于核心目标；又要通过“群众路线”深入一线，倾听市场和员工的声音，将自下而上的智慧融入决策，并赋能基层团队，让他们在执行中拥有足够的自主权和纠偏能力，形成上下联动、同频共振的组织合力。
