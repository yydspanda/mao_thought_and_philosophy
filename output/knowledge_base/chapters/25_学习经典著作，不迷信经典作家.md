# 第25章 学习经典著作，不迷信经典作家

> **摘要**：本章核心在于毛泽东在1958年针对大跃进初期出现的“左”倾冒进和教条主义倾向，通过批判性解读斯大林著作，强调破除迷信、解放思想，并提出以实践为检验真理标准，鼓励青年创新，以及在复杂矛盾中保持动态平衡的方法论。

<summary><strong>📄 点击展开/收起：本章原文全文</strong></summary>

毛主席在1958年成都会议上，围绕破除迷信，解放思想，纵论古今中外，趣味横生。我的笔记里有如下一段记录：“从古以来，创立新思想、新学派、新教派的人，都是些学问不足的年轻人。他们一眼看出新东西，同那些老古董作战，而没有不遭到反对的。马丁•路德即是如此。安眠药是德国的一个司药发明的，但遭到压迫，后来是法国人承认的。大发明家瓦特是工人出身;爱迪生是卖报的报童出身，第一个发现电的弗兰克林只读过三年书，高尔基也只读过两年书。可见不进大学也能成为大发明家、大文豪。问题在于方向对不对。总之年轻人一抓住真理就藐视那些老古董。难道历史不是这样吗？我看是这样。孔夫子是如此，耶稣是如此。他们都在年轻时即成名。释迦牟尼只19岁就创立教派，孙中山年轻时就搞革命。那时他有多少学问？学问是后头在革命中去研究、学习而丰富起来的。马克思创立无产阶级学说时年纪也很轻，30岁左右已经形成，实际上20多岁即已开始。在我国古代，贾谊死时30几岁，王弼死时24岁，李贺死时27岁，夏完淳死时17岁，都是英俊天才，惜乎死得太早了。岳飞被冤杀时才30岁。李世民起事时，18岁就当了总司令。近代的梁启超也是在20几岁，就把七八十岁的老翰林搞得望风披靡。总之，历史上总是学问少的人推翻学问多的人，总是青年胜过老年。因为青年人精力旺盛，迷信较少，顾虑少，天不怕，地不怕，敢想敢说敢干。而有学问的老年人，出了名，患得患失，顾虑多，迷信也多。毛主席还说：“我们要学习马列主义的经典著作，但不要迷信经典作家，把他们当作神，照搬而已，抄书而已。联共党史结束语六条中讲理论的一条，即说可以修改过时的结论嘛！迷信经典作家在我们脑子中产生一种镇压的力量。这种态度在学习经典作家时，就没有经典作家那种高屋建瓴、势如破竹的风格，就不可能解放思想。”毛主席的这段话非常精辟，对于读经典著作应该采取什么态度，给了我们非常深刻的教诲！<br><br>如何学习经典著作而又不迷信经典作家，1958年第一次郑州会议期间，毛主席给我们作了示范，至今记忆犹新。这次议，主要纠正混淆社会主义和共产主义界限的“左”的错误思想。11月10日清晨，大家被通知到一个小礼堂开会。一开头毛主席就以洪钟般的嗓音说，现在秀才要造反（指陈伯达在会议开始时提出要废除商品，以劳动券代替人民币等“左”的言论——作者注），你们知道不知道？今天我给大家开课，讲斯大林的《苏联社会主义经济问题》。他用一整天的时间，有的放矢逐章逐段地作了讲解。对斯大林的观点，有肯定和发挥，也有否定和商榷。例如：对于斯大林关于社会主义制度下商品生产和商品交换的必要性及其作用的论述，以及关于价值规律的论述，毛主席总的方面是肯定的，并根据中国的具体实践是有所发挥的。他以此来批评我们党内一些同志中存在着的“左”的观点。他说，我是用斯大林这个死人来压活人。斯大林对英国革命胜利后是否废除商品生产仍有保留。这种对理论问题、原则问题不武断作结论的态度是好的。但是现在我们有些同志在我国目前生产力发展水平远远落后于英国的情况下，就要废除商品生产，废除货币，这些同志岂不是像斯大林书中所说的那样“可怜的马克思主义者”么，但是在另一方面，毛主席又批评斯大林在社会主义制度下承认商品生产、商品交换、价值规律等原则问题上还是羞羞答答的。最明显的一点是，斯大林承认消费品是商品而不承认生产资料也是商品。对社会主义制度下价值规律的作用，虽然斯大林指出了并不限于商品流通范围，也扩张到生产方面，但是，价值规律是否仅仅起经济核算的计算作用，斯大林没有说清楚。<br><br>毛主席还批评斯大林对国营企业生产的生产资料和集体农庄生产的生活资料的关系没有弄清楚。为了扩大积累，不是等价交换，而是提高工业品价格，扩大剪刀差，对农民竭泽而渔。毛主席还风趣地说，在发展大城市问题上，斯大林对恩格斯是有不同意见的。恩格斯主张在一个国家内人口应相对地均衡分布，所以他不赞成发展大城市，认为弊多利少。相反，斯大林认为大城市对于发展经济利多弊少。所以他不赞成集体农庄办工业，不搞农村工业化。毛主席说，我赞成恩格斯的意见，不赞成斯大林的意见。在我国,不实行公社工业化也即农村工业化，不发展星罗棋布的中小城镇，而让农民大批大批自发地涌进大城市，这不但不利于国民经济的总体发展，也不能逐步缩小以至消灭城乡、工农差别的。<br><br>毛主席为什么能够高屋建瓴、势如破竹地解剖斯大林的著作呢？我的体会是：他对经典作家没有迷信，头脑中没有“镇压感”；他有丰富的实践经验，以实践作为检验真理的唯一标准，特别是以中国的实践来鉴别斯大林著作中的理论是非，所有这些是永远值得我们学习的。

## 🧠 深度思考与解读

## 1. 历史场域与战略意图 (Context & Intent)

本章内容主要围绕毛泽东在1958年成都会议和第一次郑州会议上的讲话展开。这一时期，中国正处于“大跃进”初期，全国上下对社会主义建设充满高涨的热情，但也伴随着严重的“左”倾冒进思想。

**时局：**

1956年苏共二十大赫鲁晓夫批判斯大林，引发了国际共产主义运动的巨大震动和思想混乱，也促使中共重新审视斯大林模式和教条主义。在国内，大跃进中出现了混淆社会主义与共产主义界限、急于过渡、否定商品生产和价值规律的“左”倾思潮。例如，文中提及陈伯达在郑州会议上提出要废除商品、以劳动券代替人民币等“左”的言论，这正是当时党内思想混乱和冒进倾向的典型体现。

**靶子：**

毛泽东的讲话主要是在批判和纠正以下具体的错误倾向：

1.  **教条主义和本本主义：** 盲目迷信经典作家和经典著作，将其奉为神明，照搬照抄，不敢越雷池一步。这种思想压制了理论创新和实践发展，导致了“镇压的力量”。
2.  **“左”倾冒进思想：** 具体表现为在经济建设中，急于求成，混淆社会主义与共产主义的界限，否定商品生产和价值规律的必要性，试图超越生产力发展阶段。
3.  **官僚主义和脱离实际：** 那些“有学问的老年人”因“患得患失，顾虑多，迷信也多”，不敢创新，脱离了生机勃勃的实践，阻碍了新思想的产生和发展。

**意图：**

毛泽东发表此文的战略目的是多方面的：

1.  **统一思想，纠正“左”倾错误：** 通过批判教条主义，为纠正大跃进中的“左”倾错误提供思想武器，特别是澄清商品生产和价值规律在社会主义阶段的地位和作用，避免经济建设脱离实际，防止因急于求成而导致更大的失误。
2.  **解放思想，鼓励创新：** 破除对权威和经典的盲目崇拜，鼓励全党特别是年轻人敢于质疑、敢于实践、敢于创新，为中国特色社会主义建设探索新路，激发全党的活力和创造力。
3.  **维护党的独立自主性：** 借批判斯大林著作中的不足，强调中国革命和建设必须立足本国实际，不能照搬外国经验，即使是社会主义阵营的“老大哥”的经验。这与苏共二十大后中共独立自主探索道路的努力一脉相承，体现了在国际共运复杂背景下，中共坚持独立自主路线的决心。

## 2. 核心方法论建模 (Methodology Modeling)

基于上述背景，毛泽东提出了以下极具洞察力的破局之道，可提炼为可复用的思维模型：

*   **[[经典批判性学习模型]] (Critical Learning Model for Classics):**
    *   **定义:** 一种对待经典理论和权威思想的系统性学习方法，强调在深入学习其精髓的同时，保持批判性思维，敢于质疑、辨析和修正过时或不符合实际的结论。
    *   **操作步骤:**
        1.  **深入研读与掌握：** 认真学习经典著作，掌握其基本原理和核心观点，这是批判的基础。
        2.  **实践检验为准：** 将经典理论与本国、本组织的具体实践相结合，以实践作为检验真理的唯一标准，而非理论本身。
        3.  **辩证分析与扬弃：** 对经典观点进行“有肯定和发挥，也有否定和商榷”的两点论分析，既不全盘肯定，也不全盘否定，而是取其精华、去其糟粕。
        4.  **破除迷信与镇压感：** 克服对经典作家和理论的盲目崇拜和“镇压感”，敢于挑战权威，解放思想，不让既有理论成为思想的桎梏。
        5.  **创造性运用与发展：** 结合实际情况，对经典理论进行创造性运用、发展和修正，形成符合自身特点的新理论和新实践。

*   **[[青年驱动创新法则]] (Youth-Driven Innovation Principle):**
    *   **定义:** 强调组织应重视并赋能年轻人，因为他们往往“精力旺盛，迷信较少，顾虑少，天不怕，地不怕，敢想敢说敢干”，是打破传统、推动创新的关键力量。
    *   **操作步骤:**
        1.  **识别与赋能：** 发现那些有新思想、敢于挑战现状的年轻人，给予他们平台、资源和信任。
        2.  **鼓励试错与创新：** 营造一个允许失败、鼓励探索的文化氛围，减少年轻人“患得患失”的顾虑，激发其创新活力。
        3.  **挑战传统权威：** 鼓励年轻人对现有规则、流程和权威观点提出质疑，促进组织内部的良性竞争和思想碰撞，打破僵化。
        4.  **实践中培养：** 认识到“学问是后头在革命中去研究、学习而丰富起来的”，将年轻人投入到实际工作中去锻炼和成长，使其理论与实践相结合。

*   **[[两条战线的斗争]] (Two-Front Struggle) - 作为[[动态决策平衡模型]]的具体应用：**
    *   **定义:** 在纠正一种主要错误倾向（如“左”倾冒进）时，必须同时警惕并防范另一种对立倾向（如“右”倾保守或对理论的盲目否定）的出现，以保持战略的平衡性和正确性。
    *   **操作步骤:**
        1.  **识别主要矛盾：** 明确当前需要优先解决的主要错误倾向（例如，郑州会议上是“左”倾冒进，否定商品生产）。
        2.  **集中火力纠正：** 针对主要矛盾，运用理论和实践进行有力批判和纠正（例如，毛主席用斯大林著作肯定商品生产来压制“左”派）。
        3.  **警惕次要矛盾转化：** 在纠正过程中，预见并防范可能出现的另一种极端倾向（例如，在肯定商品生产时，也要警惕对资本主义商品经济的盲目肯定，或对理论的过度否定）。
        4.  **保持动态平衡：** 持续评估政策实施效果，根据实际情况调整策略，避免从一个极端走向另一个极端，确保战略的灵活性和适应性。

## 3. 哲学底层逻辑 (Philosophical Root)

这些方法论深刻体现了马克思主义的认识论和辩证法精髓：

*   **实践是检验真理的唯一标准：** 毛泽东能够“高屋建瓴、势如破竹地解剖斯大林的著作”，根本在于他“有丰富的实践经验，以实践作为检验真理的唯一标准，特别是以中国的实践来鉴别斯大林著作中的理论是非”。这直接体现了马克思主义认识论的核心，即真理不是从书本中来，而是从实践中来，并最终要回到实践中去检验。对经典的批判性学习，正是将理论置于实践的熔炉中进行检验和发展，而非盲目接受。

*   **矛盾论与两点论：**
    *   **主要矛盾与次要矛盾的转化：** 在郑州会议上，毛泽东将“左”倾冒进视为当时的主要矛盾，需要集中力量纠正。但他同时又指出斯大林在某些问题上的“羞羞答答”，这体现了对次要矛盾的关注，以及预见矛盾可能转化的辩证思维。这种对矛盾的全面把握，是避免片面性和极端化的关键。
    *   **对立统一：** 毛泽东对斯大林著作的分析，既有肯定和发挥，也有否定和商榷，这正是对立统一规律的运用。他看到了斯大林理论中的合理成分和局限性，而不是简单地全盘肯定或否定。这种“[[两条战线的斗争]]”思维，本质上就是如何在对立中寻找统一、在斗争中保持平衡的辩证法，强调事物内部的矛盾性及其相互转化。

*   **认识的无限发展性：** 毛泽东强调“可以修改过时的结论”，反对“迷信经典作家”，这揭示了认识是一个不断发展、不断深化的过程，没有终极真理，只有在实践中不断接近真理。这与形而上学认为真理是一成不变的观点形成了鲜明对比，鼓励人们永不停止探索和创新。

*   **内因与外因的辩证关系：** 青年人之所以能创新，是因为他们“精力旺盛，迷信较少，顾虑少”（内因），但这种创新也需要外部环境的鼓励和支持（外因），例如毛泽东的讲话本身就是一种外部推动力。同时，中国革命的实践（内因）是检验斯大林理论（外因）正确与否的关键。这体现了内因是变化的根据，外因是变化的条件，外因通过内因起作用的辩证关系。

## 4. 现实场景迁移 (Pragmatic Application)

历史总是押韵的。在现代职场和企业管理中，我们依然能看到类似的“历史重演”和思想困境。

**场景映射：**

*   **“教条主义”与“本本主义”：** 许多企业在学习“最佳实践”（Best Practice）时，容易陷入盲目照搬的陷阱，不顾自身实际情况，导致水土不服。例如，盲目学习硅谷的扁平化管理、OKR（目标与关键成果法）或敏捷开发，却忽视了自身组织文化、人员素质和业务特点，最终效果不佳甚至适得其反。对“成功学”或“管理大师”理论的过度迷信，也属于此类，阻碍了企业根据自身情况进行创新和调整。
*   **“左”倾冒进与“右”倾保守：** 在创新转型中，一些企业可能急于求成，盲目追求“风口”，投入大量资源却缺乏深思熟虑，导致资源浪费和项目失败（“左”倾冒进）。另一些企业则因循守旧，害怕变革，对新技术、新模式视而不见，最终错失发展机遇（“右”倾保守）。这两种极端都可能导致企业战略失衡。
*   **“老古董”与“年轻人”的冲突：** 传统企业中，经验丰富但思维固化的老员工与充满活力、敢于挑战的新生代员工之间的观念冲突屡见不鲜。老员工可能因“患得患失”而阻碍创新，年轻人则可能因缺乏经验而过于激进。如何平衡和激发这两股力量，是现代组织面临的普遍挑战。

**行动建议：**

1.  **构建“批判性学习型组织”：** 鼓励团队成员对行业“最佳实践”、权威理论和内部成功经验进行**[[经典批判性学习模型]]**。在引入新方法、新工具时，不盲目崇拜，而是先进行小范围试点，结合自身实际进行调整和优化。例如，在推行新的管理制度前，组织内部应组织研讨会，鼓励不同层级、不同部门的员工提出质疑和改进意见，而不是简单地自上而下推行，确保理论与实践的结合。
2.  **激活“青年驱动创新引擎”：** 建立机制，主动识别并赋能组织内的年轻力量。例如，设立“青年创新实验室”或“挑战者项目”，让年轻人有机会提出颠覆性想法，并给予资源支持和试错空间。同时，通过导师制等方式，引导年轻人将“敢想敢干”与“扎实实践”相结合，避免空谈，真正发挥**[[青年驱动创新法则]]**的效用。
3.  **运用“动态决策平衡模型”进行战略校准：** 在制定和执行战略时，管理者要时刻警惕“[[两条战线的斗争]]”。例如，在强调创新和速度时，要警惕可能出现的盲目扩张和资源浪费；在强调稳健和风险控制时，也要警惕可能导致的僵化和错失机遇。定期进行战略复盘，评估当前主要风险是“过度激进”还是“过于保守”，并及时调整重心，确保组织在复杂环境中保持正确的航向，实现可持续发展。

### 💬 振聋发聩的金句
> 从古以来，创立新思想、新学派、新教派的人，都是些学问不足的年轻人。他们一眼看出新东西，同那些老古董作战，而没有不遭到反对的。
>
> 我们要学习马列主义的经典著作，但不要迷信经典作家，把他们当作神，照搬而已，抄书而已。
>
> 迷信经典作家在我们脑子中产生一种镇压的力量。这种态度在学习经典作家时，就没有经典作家那种高屋建瓴、势如破竹的风格，就不可能解放思想。
>
> 我是用斯大林这个死人来压活人。
>
> 他对经典作家没有迷信，头脑中没有“镇压感”；他有丰富的实践经验，以实践作为检验真理的唯一标准，特别是以中国的实践来鉴别斯大林著作中的理论是非，所有这些是永远值得我们学习的。
>
