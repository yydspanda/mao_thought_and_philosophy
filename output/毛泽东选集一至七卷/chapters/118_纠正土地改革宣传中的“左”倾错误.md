# 第118章 纠正土地改革宣传中的“左”倾错误

> **归属**：第四卷 / 第三次国内革命战争时期
> **发表时间**：一九四八年二月十一日

> **摘要**：毛泽东在1949年4月为中共中央起草指示，严厉批评党内宣传工作中出现的“左”倾错误，特别是孤立宣传贫雇农路线、唯成分论、土地改革中的急性病以及尾巴主义，强调必须纠正这些偏离统一战线和党的领导原则的错误，以确保革命的全面胜利和新政权的巩固。

最近几个月中，许多地方的通讯社和报纸，不加选择地没有分析地传播了许多包含“左”倾错误偏向的不健全的通讯或文章。例如：<br><br>一、不是宣传依靠贫雇农，巩固地联合中农，消灭封建制度的路线，而是孤立地宣传贫雇农路线。不是宣传无产阶级联合一切劳动人民、受压迫的民族资产阶级、知识分子和其他爱国分子（其中包括不反对土地改革的开明绅士），推翻帝国主义、封建主义和官僚资本主义的统治，建立中华人民共和国和人民民主政府，而是孤立地宣传所谓贫雇农打江山坐江山，或者说民主政府只是农民的政府，或者说民主政府只应该听工人和贫雇农的意见，而对中农，对独立劳动者，对民族资产阶级，对知识分子等，则一概不提。这是严重的原则性的错误。而我们的通讯社、报纸或广播电台竟将这类通讯发表。各地党委宣传部，对于此类错误竟没有任何的反映。此类宣传，在过去几个月中虽然不是普遍的，但是相当多，以致造成了一种空气，使人们误认为似乎这是正确的领导思想。甚至因为陕北广播电台播发了某些不正确的新闻，人们竟误认为这是被中央认可的意见。<br><br>二、在整党问题上，关于既反对忽视成分、又反对唯成分论的宣传，有些地区不够有力，甚至有唯成分论的错误宣传。<br><br>三、在土地改革问题上，关于既反对观望不前、又反对急性病的宣传，有些地区是抓紧了；但在许多地区却助长急性病，甚至发表赞扬急性病的东西。在领导者和群众的关系问题上，关于既反对命令主义、又反对尾巴主义的宣传，有些地区是注意了；但在许多地区却错误地强调所谓“群众要怎样办就怎样办”，迁就群众中的错误意见。甚至对于并非群众的、而只是少数人的错误意见，也无批判地接受。否定了党的领导作用，助长了尾巴主义。<br><br>四、在工商业和工人运动的方针上，对于某些解放区存在着的严重的“左”的倾向，或者加以赞扬，或者熟视无睹。<br><br>总之，过去几个月的宣传工作，正确地反映和指导了战争、土地改革、整党、生产、支援前线这些伟大斗争，帮助了这些斗争取得了伟大成绩，并且在宣传工作中占着主要成分，这是必须首先承认的。但是也必须看到一些错误缺点。其特点就是过左。其中有些是完全违背马克思列宁主义原则立场和完全脱离中央路线的。望各中央局、中央分局及其宣传部，新华总社和各地总分社，以及各地报纸的工作同志们，根据马克思列宁主义原则和中央路线，对过去几个月的宣传工作，加以检查，发扬成绩，纠正错误，务使对于战争、土地改革、整党、工人运动这些伟大的斗争，对于这一整个反帝反封建的革命，保障其获得胜利。这种检查，责成各地党委宣传部和新华总社负主要责任，并将结果用自己的名义写一个政策性的报告给中央宣传部。<br><br>注　　释<br><br>*<br><br>这是毛泽东为中共中央起草的对党内的指示。

## 🧠 深度思考与解读

## 1. 历史场域与战略意图 (Context & Intent)
这篇指示发表于1949年4月25日，正值中国人民解放军在解放战争中取得决定性胜利，即将建立全国政权的关键历史节点。当时，解放军已渡过长江，国民党政权行将覆灭，中国共产党正面临着从革命战争转向全国执政的重大转变。在这一背景下，党内部分同志在宣传和政策执行中出现了“左”倾错误倾向，毛泽东敏锐地感知到这些“风起于青萍之末”的风险。

当时面临的紧迫危机是：
1.  **统一战线被破坏的风险**：部分宣传工作“孤立地宣传贫雇农路线”，忽视了巩固地联合中农、争取民族资产阶级、知识分子和开明绅士的重要性，甚至提出“贫雇农打江山坐江山”的口号，这严重偏离了党在民主革命阶段的统一战线策略，可能导致新政权失去广泛的社会基础，激化不必要的矛盾。
2.  **党内思想混乱与政策执行偏差**：在整党中出现“唯成分论”，在土地改革中助长“急性病”，在领导者与群众关系上强调“群众要怎样办就怎样办”的“尾巴主义”，这些都反映出党内对马克思列宁主义原则和中央路线理解的偏差，可能导致政策执行的简单化、粗暴化，损害党的形象和群众利益。
3.  **党的领导作用被削弱**：无论是“尾巴主义”还是对“左”倾错误的熟视无睹，都意味着党的领导核心作用被弱化，无法有效引导革命进程和国家建设。

毛泽东发表此文的战略目的是：
*   **统一全党思想**：纠正党内尤其是宣传战线上的“左”倾错误，确保全党在即将到来的全国执政中，能够坚定不移地执行正确的统一战线政策和各项方针。
*   **巩固革命胜利成果**：通过纠正偏激的政策倾向，争取和团结一切可以团结的力量，为新中国的建立和初期建设奠定坚实的社会基础，避免因内部错误而引发新的动荡。
*   **维护党的领导权威**：重申党的领导作用，强调在群众路线中，党既要倾听群众意见，又要发挥教育和引导作用，防止盲目迁就群众中的错误倾向，确保党的路线方针政策的正确贯彻。

## 2. 核心方法论建模 (Methodology Modeling)
针对上述危机，毛泽东提出了以下破局方法，可抽象为现代管理模型：

1.  **[[动态联盟策略]] 与 [[深度利益相关者分析]]**：
    *   **问题**：宣传中“孤立地宣传贫雇农路线”，忽视了中农、民族资产阶级、知识分子和开明绅士等其他社会力量。
    *   **模型操作**：在制定任何战略或政策时，必须进行全面的利益相关者分析，识别所有可能影响或被影响的群体。理解他们的利益诉求、政治态度和潜在作用，并据此构建一个广泛的、动态的联盟。避免将某一群体绝对化，而是根据主要矛盾和次要矛盾的转化，灵活调整联盟范围和策略。例如，在革命的不同阶段，对民族资产阶级和开明绅士的态度是争取和团结，而非排斥。

2.  **[[动态决策平衡模型]]**：
    *   **问题**：在整党中出现“唯成分论”，在土地改革中助长“急性病”，在领导与群众关系中出现“尾巴主义”，这些都是偏离中庸之道的极端倾向。
    *   **模型操作**：认识到任何政策或行动都可能存在“左”和“右”两种极端。正确的决策并非简单地选择其中之一，而是在对立统一中寻找动态平衡点。这要求领导者具备高度的辩证思维，既要反对“观望不前”，也要反对“急性病”；既要反对“忽视成分”，也要反对“唯成分论”；既要反对“命令主义”，也要反对“尾巴主义”。通过持续的实践反馈和理论指导，不断校准决策，避免滑向任何一端。

3.  **[[组织韧性构建模型]] 与 [[组织纯洁与韧性建设]]**：
    *   **问题**：宣传部门对“左”倾错误熟视无睹，甚至发表赞扬性内容，否定了党的领导作用。
    *   **模型操作**：强调党组织（或领导核心）必须保持高度的警惕性和自我纠错能力。通过定期的内部检查、批评与自我批评，及时发现并纠正组织内部出现的错误倾向和不健康空气。这包括对思想路线的检查、对政策执行的监督，以及对干部队伍的教育和整顿。确保组织始终能够坚持正确的政治方向，发挥其核心领导作用，并提升其在复杂环境下的适应性和抗风险能力。

4.  **[[基于实践的决策]]**：
    *   **问题**：宣传内容脱离中央路线和马克思列宁主义原则，甚至迁就群众中的错误意见。
    *   **模型操作**：强调所有决策和宣传工作必须以马克思列宁主义原则和中央路线为指导，并结合具体实践进行检验。反对脱离理论指导的盲目行动，也反对脱离实际的教条主义。在听取群众意见时，要进行分析和批判，不能无原则地迁就，而是要用正确的思想去引导和教育群众，确保决策的科学性和有效性。

## 3. 哲学底层逻辑 (Philosophical Root)
毛泽东的这些方法论深刻体现了马克思主义的辩证唯物主义和历史唯物主义。

*   **矛盾论与两点论**：文章的核心是运用矛盾分析法来识别和解决党内存在的“左”倾错误。毛泽东指出，革命的胜利需要解决主要矛盾（推翻帝国主义、封建主义和官僚资本主义），这就要求团结一切可以团结的力量，将次要矛盾（如民族资产阶级与无产阶级的矛盾）暂时置于服从地位。而“左”倾错误恰恰混淆了主要矛盾与次要矛盾，将统一战线中的盟友推向对立面，这是对矛盾论的根本性误读。他强调的“既反对…又反对…”的表述，正是典型的两点论思维，即在认识和处理问题时，要全面看问题，避免片面性，在对立中寻找统一，把握事物的度。

*   **认识论与实践论**：文章批评的“孤立地宣传”、“助长急性病”、“迁就错误意见”等，都反映了认识上的片面性和主观性。毛泽东强调，正确的认识来源于对客观实际的深入调查研究，并以马克思列宁主义的科学理论为指导。脱离实际的教条主义和脱离理论指导的经验主义（或称“尾巴主义”）都是错误的认识论。他要求各级党委和宣传部门“加以检查，发扬成绩，纠正错误”，正是通过实践（检查工作）来检验和修正认识（宣传路线），从而实现理论与实践的统一。

*   **辩证法与度的把握**：无论是对统一战线中各阶级的态度，还是在政策执行中的“急”与“缓”、“领导”与“群众”的关系，毛泽东都强调要把握好“度”。“过左”就是失去了这个度，导致事物的性质发生不利于革命的转化。他通过纠正这些偏颇，力图使党的工作回到正确的轨道，体现了对事物发展规律的深刻理解和运用。

## 4. 现实场景迁移 (Pragmatic Application)
历史总是押韵的。在现代职场和企业管理中，我们依然能看到类似的“历史重演”：

*   **“孤立地宣传”与“唯成分论”**：在企业中，这可能表现为某个部门（如研发部）过度强调自身技术的重要性，而忽视市场、销售、运营等其他部门的贡献和需求，导致产品与市场脱节。或者在团队建设中，过分强调某一类人才（如技术精英）的作用，而忽视了其他背景（如市场营销、客户服务）员工的价值，造成内部隔阂和人才流失。这类似于“孤立地宣传贫雇农路线”，未能构建一个广泛的“企业统一战线”。
*   **“急性病”与“尾巴主义”**：在项目管理中，这可能表现为为了追求短期业绩或迎合少数强势客户的需求，而盲目压缩开发周期、牺牲产品质量，导致项目烂尾或后期维护成本剧增（“急性病”）。或者，管理者缺乏主见，对团队中少数人的错误意见不加批判地接受，甚至被其裹挟，最终导致项目偏离战略方向（“尾巴主义”）。

作为管理者或专业人士，我们可以应用本章智慧：
1.  **构建“企业统一战线”，实施 [[深度利益相关者分析]] 与 [[动态联盟策略]]**：在制定企业战略或推动项目时，要超越部门壁垒，识别并分析所有内外部利益相关者（员工、客户、股东、供应商、合作伙伴、监管机构等）。理解他们的核心利益、潜在贡献和风险，并据此构建一个广泛的、动态的合作联盟。例如，在推出新产品时，不仅要听取研发和市场部门的意见，还要充分考虑销售团队的反馈、客户服务部门的痛点以及供应链的承受能力，确保产品策略的全面性和落地性。
2.  **践行 [[动态决策平衡模型]]，警惕“左右摇摆”的极端倾向**：管理者在决策时要警惕两种极端：一是“教条主义”或“路径依赖”，固守旧有模式不思变；二是“盲目激进”或“短期主义”，为追求短期效果而采取不计后果的行动。应在“稳健”与“创新”、“效率”与“公平”之间寻找动态平衡点。例如，在组织变革中，既要避免因循守旧，也要防止“一刀切”的激进改革，而应采取试点先行、逐步推广的策略，并根据反馈及时调整。
3.  **强化 [[组织韧性构建模型]]，坚持“领导力与群众路线”的辩证统一**：领导者要明确自身在战略指引、资源分配和文化塑造上的核心作用，不能将“民主”异化为“尾巴主义”。同时，要建立有效的机制倾听基层声音，鼓励员工提出意见和建议，但最终的决策权和方向把控权仍需由领导层承担。这要求领导者既要有开放的心态，又能坚守原则，在吸收群众智慧的同时，用清晰的战略目标和价值观来引导团队，纠正偏差，确保组织在复杂环境中保持战略定力和执行力。
