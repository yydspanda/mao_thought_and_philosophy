# 第46章 团结一切抗日力量，反对反共顽固派

> **归属**：第二卷 / 抗日战争时期（上）
> **发表时间**：一九四〇年二月一日

> **摘要**：本章选自1940年毛泽东在延安讨汪大会上的演说。面对汪精卫叛国与国民党顽固派掀起反共高潮的双重危机，毛泽东提出了“团结一切抗日力量，反对反共顽固派”的战略方针，通过重构“统一”的定义，并以延安为“政治样板房”进行降维打击，展示了如何在统一战线中通过斗争求团结的高超博弈艺术。

我们延安的各界人民今天在这里开会，为了什么呢？为了声讨卖国贼汪精卫<br><br>〔1〕<br><br>，又是为了团结一切抗日力量，反对反共顽固派。<br><br>我们共产党人屡次指出，日本帝国主义的灭华方针是坚决的。不管日本掉换什么内阁，它的灭亡中国把中国变为殖民地的基本方针是不会变更的。中国亲日派大资产阶级的政治代表汪精卫，看了这种情形，吓得发疯，跪倒在日本面前，订立了日汪卖国条约，把中国出卖给日本帝国主义。他还要成立傀儡政府，和抗日政府相对立；还要成立傀儡军队，和抗日军队相对立。他对于反蒋近来不大提了，据说已经改为“联蒋”。反共是日汪的主要目的。他们知道共产党抗日最彻底，国共合作则力量更大，他们就用全力分裂国共合作，使两党各自孤立，最好是两党打起来。这样，他们就利用国民党内部的顽固派，到处放火。在湖南就闹平江惨案<br><br>〔2〕<br><br>，在河南就闹确山惨案<br><br>〔3〕<br><br>，在山西就闹旧军打新军<br><br>〔4〕<br><br>，在河北就闹张荫梧打八路军<br><br>〔5〕<br><br>，在山东就闹秦启荣打游击队<br><br>〔6〕<br><br>，在鄂东就闹程汝怀杀死五六百个共产党员<br><br>〔7〕<br><br>，在陕甘宁边区就闹内部的“点线工作”<br><br>〔8〕<br><br>、外部的“封锁工作”，并且还准备着军事进攻<br><br>〔9〕<br><br>。此外，又逮捕了一大批进步青年送进集中营<br><br>〔10〕<br><br>；又雇请玄学鬼张君劢<br><br>〔11〕<br><br>提出取消共产党、取消陕甘宁边区、取消八路军新四军的反动主张，雇请托洛茨基分子叶青等人做文章骂共产党。所有这些，无非是要破坏抗日的局面，使全国人民都当亡国奴。<br><br>这样，汪精卫派和国民党的反共顽固派两家里应外合，把时局闹得乌烟瘴气了。<br><br>许多人看了这种情形，非常气愤，就以为抗日没有希望了，国民党都是坏人，都应当反对。我们必须指出，气愤是完全正当的，哪有看了这些严重情形而不气愤的呢？但是抗日仍然是有希望的，国民党里面也不都是坏人。对于各部分的国民党人，应当采取不同的政策。对于那些丧尽天良的坏蛋，对于那些敢于向八路军新四军阵地后面打枪的人，对于那些敢于闹平江惨案、确山惨案的人，对于那些敢于破坏边区的人，对于那些敢于攻打进步军队、进步团体、进步人员的人，我们是决不能容忍的，是必定要还击的，是决不能让步的。因为这类坏蛋，已经丧尽天良，当一个民族敌人深入国土的时候，他们还要闹磨擦，闹惨案，闹分裂。不管他们心里怎样想，他们是在实际上帮助了日本和汪精卫，或者有些人本来就是暗藏的汉奸。对于这些人，如果不加以惩罚，我们就是犯错误，就是纵容汉奸国贼，就是不忠实于民族抗战，就是不忠实于祖国，就是纵容坏蛋来破裂统一战线，就是违背了党的政策。但是这种给投降派和反共顽固派以打击的政策，全是为了坚持抗日，全是为了保护抗日统一战线。因此，我们对于那些忠心抗日的人，对于一切非投降派、非反共顽固派的人们，对于这样的国民党员，是表示好意的，是团结他们的，是尊重他们的，是愿意和他们长期合作以便把国家弄好的。谁如果不这样做，谁也就是违背了党的政策。<br><br>这就是我们党的两条政策：一方面，团结一切进步势力，团结一切忠心抗日的人，这是一条政策；一方面，反对一切丧尽天良的坏蛋，反对那些投降派和反共顽固派，这是又一条政策。我们党的这些政策，为了达到一个目的，这就是力争时局好转，战胜日本。我们共产党和全国人民的任务，就是团结一切抗日的进步的势力，抵抗一切投降的倒退的势力，力争时局的好转，挽救时局的逆转。这就是我们的根本方针。我们决不悲观失望，我们是乐观的。我们不怕任何投降派和反共顽固派的进攻，我们一定要粉碎他们，我们也一定能够粉碎他们。中华民族的解放是一定的，中国决不会亡国。中国的进步是一定的，倒退只是暂时的现象。<br><br>我们今天开会还要向全国人民表明一种态度，这就是为了抗日，全国人民的团结和进步是必要的。有些人单单强调了抗日，但不愿意强调团结和进步，甚至完全不提团结和进步，这是不对的。没有真正的、坚强的团结，没有迅速的、切实的进步，怎能坚持抗日？国民党的反共顽固派强调统一，但是他们的所谓“统一”，乃是假统一，不是真统一；乃是不合理的统一，不是合理的统一；乃是形式主义的统一，不是实际的统一。他们高唤统一，却原来是要取消共产党、八路军、新四军和陕甘宁边区，说有共产党、八路军、新四军和边区存在，中国就不统一，他们要把全国一切都化为国民党；不但继续他们的一党专政，而且还要扩大他们的一党专政。如果是这样，那还有什么统一呢？老实说，过去如果没有共产党、八路军、新四军和陕甘宁边区真心实意地出来主张停 ## 🧠 深度思考与解读

## 1. 历史场域与战略意图：破除“假统一”的迷雾

### 时局背景：内忧外患的至暗时刻
这篇文章发表于1940年2月1日。当时的中国正处于抗日战争的相持阶段，局势极度凶险：
*   **外部**：大汉奸汪精卫已于1939年底与日本秘密签订卖国条约，并筹备伪政府，民族危机加深。
*   **内部**：国民党内的顽固派（蒋介石集团）发动了**“第一次反共高潮”**。他们一方面在军事上制造了平江惨案、确山惨案，围攻八路军新四军；另一方面在政治上大造舆论，喊出“一个主义、一个政党、一个领袖”的口号，企图借“统一”之名，行“取消共产党”之实。

### 战略意图：以攻为守，夺取政治解释权
毛泽东此文的意图绝非简单的“抗议”，而是一场精准的**政治反击战**：
1.  **切割敌我**：明确将汪精卫（汉奸）与国民党顽固派（反共分子）和国民党进步人士（盟友）进行切割，避免四面树敌。
2.  **揭露阴谋**：撕破顽固派“统一”的伪装，指出他们的统一是“亡国的统一”，是让进步向落后投降。
3.  **确立标准**：通过展示延安的清明政治，确立中共在抗战中的道德与政治领导权。

## 2. 核心方法论建模：竞合博弈与标准重塑

针对国民党“假统一、真反共”的策略，毛泽东构建了一套极具杀伤力的反制模型：

### 模型一：[[动态竞合博弈模型]] (Dynamic Co-opetition Model)
面对国民党这一复杂的“盟友”，毛泽东拒绝了非黑即白的二元思维，而是采取了**“两手对两手”**的策略：
*   **团结维度**：对一切抗日力量、非顽固派，表示好意，长期合作。
*   **斗争维度**：对“丧尽天良的坏蛋”（制造摩擦者），坚决还击，“决不让步”。
*   **操作法则**：**以斗争求团结**。如果一味退让，只会导致统一战线破裂；只有把顽固派打痛了，他们才肯坐下来谈团结。

### 模型二：[[话语权重构法则]] (Discourse Reframing Law)
国民党当时挥舞着“统一”的大棒（道德高地）来打压中共。毛泽东没有否定“统一”，而是**重新定义了“统一”的KPI**：
*   **对手逻辑**：统一 = 听国民党的话 = 取消边区。
*   **毛泽东逻辑**：统一 = 抗战 + 团结 + 进步。
*   **破局点**：既然你要统一，那就必须建立在“进步”的基础上。如果你的统一是倒退（贪污、腐败、投降），那就是“假统一”。通过这一逻辑转换，毛泽东瞬间将被告席变成了原告席。

### 模型三：[[政治样板房策略]] (Political Prototype Strategy)
这是本章最精彩的“产品经理”思维。毛泽东列举了陕甘宁边区的“十个没有”（无贪官、无娼妓、无发国难财等），将延安打造为**“政治样板房”**。
*   **对比营销**：将延安的“清明”与国统区的“乌烟瘴气”进行并列对比。
*   **逻辑推演**：既然延安是全国最进步的地方，那么“统一”就应该是全国向延安看齐，而不是延安向落后看齐。这直接瓦解了国民党要求“取消边区”的合法性。

## 3. 哲学底层逻辑：对立统一与实事求是

*   **矛盾论的精髓**：毛泽东深刻处理了**主要矛盾（中日民族矛盾）**与**次要矛盾（国共阶级矛盾）**的关系。他指出，反共顽固派制造摩擦（次要矛盾激化）实际上是在帮助日本（主要矛盾）。因此，打击顽固派正是为了维护抗日大局，体现了**“有理、有利、有节”**的辩证法。
*   **两点论与重点论**：在看待国民党时，既看到其抗日的一面（需要团结），也看到其反动的一面（需要斗争）。不因抗日而放弃阶级斗争，也不因阶级斗争而分裂抗日大局。
*   **实践是检验真理的标准**：通过列举延安治理的实际成果（“十个没有”），用铁一般的事实反击国民党的抽象理论攻击，这是唯物主义认识论的胜利。

## 4. 现实场景迁移：职场与管理启示

### 场景一：应对职场中的“道德绑架”与“假协同”
在现代企业中，常有部门打着“公司大局”、“统一管理”或“协同”的旗号，试图收编你的团队或抢夺资源，实则效率低下。
*   **行动建议**：应用**[[话语权重构法则]]**。不要直接拒绝“协同”，而要重新定义协同的标准。例如：“我们非常支持统一管理，但统一管理的目标应该是**效率提升**和**业绩增长**。如果合并后导致响应变慢、业绩下滑，这就是‘形式主义的统一’。我们建议先以‘业绩增长’为基准，保留目前的敏捷模式。”

### 场景二：打造团队的“护城河”
当你的团队面临外部质疑或被边缘化的风险时，单纯的辩解是苍白的。
*   **行动建议**：应用**[[政治样板房策略]]**。将你的团队打造成公司内部的“延安”。建立“十个没有”式的独特文化（如：没有推诿、没有无效会议、没有办公室政治）。当你的团队产出最高、风气最正时，你就拥有了“落后向进步看齐”的底气，任何试图拆解你团队的行为都会显得不合法。

### 场景三：处理复杂的合作伙伴关系
在商业联盟中，盟友可能同时也是竞争对手（竞合关系）。
*   **行动建议**：应用**[[动态竞合博弈模型]]**。不要对合作伙伴抱有幻想，也不要全面破裂。要清晰界定底线：在共同做大市场（抗日）上全力配合；在对方试图侵犯核心利益（反共）时，必须坚决反击，打痛对方，才能赢得对方的尊重和长期的合作。
